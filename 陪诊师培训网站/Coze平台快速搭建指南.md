# 陪诊师培训网站 - Coze平台快速搭建指南

## 一、前置准备

### 1.1 账号注册
1. 访问 [扣子官网](https://coze.cn)
2. 点击「立即注册」
3. 选择「个人账号」或「企业账号」
4. 完成手机号验证和实名认证

### 1.2 工作空间创建
1. 登录后进入控制台
2. 点击「创建工作空间」
3. 填写工作空间名称（如：陪诊师培训）
4. 选择合适的工作空间类型

---

## 二、应用创建流程

### 步骤1：创建培训应用

1. 进入工作空间
2. 点击左侧菜单「应用」
3. 点击「创建应用」
4. 选择应用类型：
   - **H5应用**：适合移动端网页访问
   - **小程序**：适合微信小程序发布
5. 填写应用信息：
   - 应用名称：陪诊师在线培训平台
   - 应用描述：专业的陪诊师在线培训与认证平台
   - 应用图标：上传品牌logo

### 步骤2：配置页面导航

在应用编辑器中创建以下页面结构：

```
首页
├── 登录/注册
├── 个人中心
│   ├── 学习档案
│   ├── 我的课程
│   ├── 考试记录
│   └── 我的证书
├── 课程中心
│   ├── 课程分类
│   │   ├── 基础理论
│   │   ├── 服务规范
│   │   ├── 医疗常识
│   │   ├── 急救技能
│   │   └── 法律法规
│   └── 课程详情
├── 学习中心
│   ├── 视频播放器
│   └── 图文阅读
└── 考试中心
    ├── 考试说明
    ├── 在线答题
    └── 成绩查询
```

## 三、核心功能配置

### 3.1 会员登录流程

```
触发条件：用户点击「登录」按钮

1. 弹窗选择登录方式
   ├── 手机号+验证码
   └── 账号+密码

2. 手机号登录流程
   ├── 输入手机号
   ├── 点击获取验证码
   ├── 输入验证码
   └── 验证通过后写入数据库

3. 登录成功
   ├── 跳转到首页
   └── 显示用户信息
```

### 3.2 课程学习流程

```
用户进入课程详情页

1. 显示课程信息
   ├── 封面图
   ├── 课程标题
   ├── 课程描述
   └── 已学习进度

2. 点击「开始学习」
   ├── 判断是否登录
   │   ├── 未登录 → 跳转登录页
   │   └── 已登录 → 继续
   ├── 记录学习开始
   └── 进入播放器/阅读器

3. 学习过程中
   ├── 每30秒更新一次进度
   ├── 记录播放位置
   └── 累计学习时长

4. 完成学习
   ├── 视频播放到90%以上
   ├── 或图文滑动到底部
   ├── 更新进度为100%
   └── 标记章节完成
```

### 3.3 考试流程

```
用户进入考试中心

1. 资格考试校验
   ├── 检查是否完成所有课程
   │   ├── 未完成 → 提示：请先完成所有课程
   │   └── 已完成 → 继续
   ├── 检查考试次数
   │   ├── 已达上限 → 提示：考试次数已用完
   │   └── 未达上限 → 继续
   └── 间隔时间检查（如有）

2. 开始考试
   ├── 生成试卷（随机抽题）
   ├── 显示考试说明
   └── 倒计时开始

3. 答题过程
   ├── 显示题目和选项
   ├── 点击选项作答
   ├── 切换下一题
   └── 保存答题进度

4. 提交试卷
   ├── 确认交卷
   ├── 自动评分
   ├── 显示成绩
   └── 判定是否通过
```

### 3.4 证书生成流程

```
触发条件：考试成绩通过

1. 资格校验
   ├── 所有课程完成 ✓
   ├── 考试成绩≥80分 ✓
   └── 实名认证 ✓

2. 生成证书
   ├── 生成唯一证书编号
   ├── 写入证书表
   └── 生成防伪二维码

3. 证书展示
   ├── 显示证书预览
   ├── 提供下载按钮
   └── 支持打印
```

## 五、考试题库设计

### 5.1 题型分布

| 单选题 | 30题 | 2分/题 | 共60分 |

| **合计** | **50题** | **100分** | |

### 5.2 题库分类

| 分类 | 题目数量 | 说明 |
|-----|---------|------|
| 基础理论 | 80题 | 职业认知、礼仪、沟通等 |
| 服务规范 | 60题 | 服务流程、标准等 |
| 医疗常识 | 100题 | 疾病、检查、用药等 |
| 急救技能 | 50题 | CPR、急救处理等 |
| 法律法规 | 40题 | 法规、隐私、纠纷等 |
| **合计** | **330题** | |

### 5.3 题目示例

#### 单选题示例
```
题目：陪诊师在引导患者就诊时，以下做法正确的是：
A. 根据挂号顺序直接带患者到诊室门口等候
B. 提前了解当日出诊医生信息，协助患者选择
C. 代替患者与医生沟通病情
D. 在患者与医生沟通时插话补充

答案：B
```

#### 多选题示例
```
题目：以下哪些情况陪诊师应及时联系医护人员？
A. 患者突然晕倒
B. 患者出现呼吸困难
C. 患者要求复印病历
D. 患者情绪激动可能出现自伤行为

答案：ABD
```

#### 判断题示例
```
题目：陪诊师可以代替患者签署任何医疗知情同意书。（×）

答案：错误
解析：陪诊师只能协助患者了解知情同意书内容，签署必须由患者本人或法定代理人完成。
```

## 七、发布上线

### 7.1 H5网页发布

1. 进入应用编辑器
2. 点击右上角「发布」
3. 选择「H5网页」平台
4. 配置域名（如已备案）
5. 提交审核
6. 审核通过后获取访问链接

### 7.2 微信小程序发布

1. 准备材料：
   - 小程序管理员微信
   - AppID（需提前申请）
   - 小程序名称、图标
   - 类目选择：教育-在线课程

2. 发布流程：
   - 在Coze中导出小程序代码包
   - 登录微信公众平台
   - 上传代码包
   - 提交审核
   - 审核通过后发布

### 7.3 注意事项

| 事项 | 说明 |
|-----|------|
| 域名备案 | 如需绑定自定义域名，需完成ICP备案 |
| 微信认证 | 小程序需完成微信认证才能调用部分API |
| 内容合规 | 课程内容需符合平台规范，不得含有违规内容 |
| 隐私政策 | 需提供用户隐私政策和用户协议 |
