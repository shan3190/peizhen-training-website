# 陪诊师培训网站 - 需求分析与功能设计方案

---

## 一、项目概述

### 1.1 项目背景

随着医疗陪诊服务行业的快速发展，专业陪诊师的需求日益增长。为提升陪诊师的专业素质和服务水平，需要搭建一套完整的在线培训系统，实现培训流程的数字化、规范化管理。

### 1.2 项目目标

- 建立完善的学员管理体系，实现培训学员的全生命周期管理
- 构建丰富的在线课程体系，支持视频、图文等多种教学形式
- 实现学习进度可视化追踪，确保培训质量
- 搭建在线考试系统，支持自动评分和成绩管理
- 提供培训证书在线生成和下载功能

### 1.3 目标用户

| 用户角色 | 描述 |
|---------|------|
| 学员 | 参加陪诊师培训的人员，通过平台学习课程、参加考试、获取证书 |
| 培训管理员 | 管理课程内容、学员信息、考试成绩，监督培训进度 |
| 客服人员 | 解答学员疑问，处理日常事务 |

## 二、功能需求分析

### 2.1 会员系统

#### 2.1.1 学员注册与登录
- 支持手机号+验证码注册/登录
- 支持账号密码登录
- 支持第三方登录（微信）
- 注册信息包括：姓名、手机号、身份证号、头像

#### 2.1.2 学员信息管理
- 个人资料编辑（姓名、手机号、头像）
- 密码修改
- 实名认证信息管理
- 学习档案查看

#### 2.1.3 学员学习档案
- 累计学习时长
- 已完成课程数
- 考试通过记录
- 获得的证书列表

### 2.2 课程系统

#### 2.2.1 课程分类管理
| 课程类别 | 课程数量 | 说明 |
|---------|---------|------|
| 基础理论 | 5-8门 | 陪诊服务概述、职业礼仪、沟通技巧等 |
| 服务规范 | 4-6门 | 服务流程标准化、应急预案处理等 |
| 医疗常识 | 6-8门 | 常见疾病知识、用药常识、检验检查解读等 |
| 急救技能 | 3-5门 | 心肺复苏、止血包扎、意外伤害处理等 |
| 法律法规 | 3-4门 | 医疗相关法规、隐私保护、纠纷处理等 |

#### 2.2.2 课程内容形式
- **视频课程**：支持在线播放、倍速播放、进度记忆
- **图文教材**：支持富文本排版、图片展示
- **课后练习**：每个课程配套章节测验

#### 2.2.3 课程详情页
- 课程封面图、标题、简介
- 课程时长、章节数
- 学习人数、评分
- 课程目录（章节列表）
- 立即学习/继续学习按钮

### 2.3 进度管理系统

#### 2.3.1 学习进度追踪
- 单个课程进度百分比（观看视频进度 + 图文阅读标记）
- 章节完成状态（未学/在学/已完成）
- 章节内视频播放进度
- 学习时间点记录（断点续学）

#### 2.3.2 整体学习看板
- 整体学习进度（百分比）
- 各分类课程完成情况
- 累计学习时长
- 连续学习天数

#### 2.3.3 进度规则
- 视频需播放至90%以上视为完成
- 图文内容需滑动到底部视为阅读完成
- 所有课程进度达100%才能参加考试

### 2.4 考试系统

#### 2.4.1 题库管理
- 题目类型：单选题、多选题、判断题
- 题库分类：按课程分类或综合题库
- 题目管理：添加、编辑、删除、批量导入

#### 2.4.2 试卷配置
- 考试名称、时间限制、及格分数
- 题目数量、分值配置
- 随机抽题规则或固定试卷

#### 2.4.3 在线答题
- 考试条件校验（是否完成所有课程）
- 倒计时显示
- 答题进度保存
- 交卷确认

#### 2.4.4 成绩管理
- 自动评分系统
- 考试成绩展示（分数、错题分析）
- 考试记录查询
- 考试次数限制（默认3次）

#### 2.4.5 考试规则
| 规则项 | 配置值 |
|-------|-------|
| 及格分数 | 80分 |
| 考试次数限制 | 3次 |
| 考试时长 | 60分钟 |
| 题目数量 | 50题 |
| 重新考试间隔 | 24小时 |

### 2.5 证书系统

#### 2.5.1 证书生成条件
- 所有课程学习进度达100%
- 考试成绩≥80分
- 实名认证信息完整

#### 2.5.2 证书内容
- 证书编号（唯一标识）
- 学员姓名
- 身份证号（脱敏显示）
- 培训项目名称
- 培训完成日期
- 证书有效期（如有）
- 颁发机构信息
- 防伪二维码/验证码

#### 2.5.3 证书展示与下载
- 证书在线预览
- PDF格式下载
- 打印功能

## 三、技术方案选型建议

### 3.1 方案对比

| 维度 | 方案一：Coze扣子平台 | 方案二：传统开发 |
|-----|---------------------|------------------|
| 开发周期 | 5-10个工作日 | 30-60个工作日 |
| 开发成本 | 低（平台费用） | 高（人力+服务器+维护） |
| 技术门槛 | 低，无需编程基础 | 高，需要专业开发团队 |
| 功能扩展 | 受限于平台能力 | 高度可定制 |
| 数据控制 | 数据存储在平台 | 完全自主控制 |
| 适合场景 | 快速验证 MVP | 长期运营、企业级需求 |

### 3.2 推荐方案

**优先推荐：Coze扣子平台**

理由：
1. 开发周期短，可快速上线验证
2. 内置AI能力，可提供智能客服、个性化学习推荐
3. 支持发布为H5网页，微信内直接访问
4. 维护成本低，无需专业运维团队
5. 可与微信生态无缝集成

### 5.2 数据库设计

```sql
-- 学员表
CREATE TABLE `students` (
  `id` VARCHAR(36) PRIMARY KEY,
  `name` VARCHAR(50) NOT NULL COMMENT '姓名',
  `phone` VARCHAR(20) NOT NULL UNIQUE COMMENT '手机号',
  `password` VARCHAR(255) NOT NULL COMMENT '密码',
  `id_card` VARCHAR(20) COMMENT '身份证号',
  `avatar` VARCHAR(500) COMMENT '头像URL',
  `status` TINYINT DEFAULT 1 COMMENT '状态:0禁用1正常',
  `created_at` DATETIME DEFAULT CURRENT_TIMESTAMP,
  `updated_at` DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
);

-- 课程分类表
CREATE TABLE `course_categories` (
  `id` VARCHAR(36) PRIMARY KEY,
  `name` VARCHAR(50) NOT NULL COMMENT '分类名称',
  `icon` VARCHAR(255) COMMENT '图标',
  `sort` INT DEFAULT 0 COMMENT '排序',
  `created_at` DATETIME DEFAULT CURRENT_TIMESTAMP
);

-- 课程表
CREATE TABLE `courses` (
  `id` VARCHAR(36) PRIMARY KEY,
  `category_id` VARCHAR(36) NOT NULL COMMENT '分类ID',
  `title` VARCHAR(200) NOT NULL COMMENT '课程标题',
  `cover` VARCHAR(500) COMMENT '封面图',
  `description` TEXT COMMENT '课程描述',
  `type` ENUM('video', 'article') DEFAULT 'video' COMMENT '内容类型',
  `content` TEXT COMMENT '内容(富文本或视频URL)',
  `duration` INT DEFAULT 0 COMMENT '时长(分钟)',
  `sort` INT DEFAULT 0 COMMENT '排序',
  `status` TINYINT DEFAULT 1 COMMENT '状态',
  `created_at` DATETIME DEFAULT CURRENT_TIMESTAMP,
  FOREIGN KEY (`category_id`) REFERENCES `course_categories`(`id`)
);

-- 章节表
CREATE TABLE `chapters` (
  `id` VARCHAR(36) PRIMARY KEY,
  `course_id` VARCHAR(36) NOT NULL COMMENT '课程ID',
  `title` VARCHAR(200) NOT NULL COMMENT '章节标题',
  `type` ENUM('video', 'article') NOT NULL COMMENT '内容类型',
  `content` TEXT COMMENT '内容(富文本或视频URL)',
  `duration` INT DEFAULT 0 COMMENT '时长(分钟)',
  `sort` INT DEFAULT 0 COMMENT '排序',
  FOREIGN KEY (`course_id`) REFERENCES `courses`(`id`)
);

-- 学习进度表
CREATE TABLE `learning_progress` (
  `id` VARCHAR(36) PRIMARY KEY,
  `student_id` VARCHAR(36) NOT NULL COMMENT '学员ID',
  `chapter_id` VARCHAR(36) NOT NULL COMMENT '章节ID',
  `progress` DECIMAL(5,2) DEFAULT 0 COMMENT '进度百分比',
  `total_duration` INT DEFAULT 0 COMMENT '累计学习时长(秒)',
  `completed` TINYINT DEFAULT 0 COMMENT '是否完成',
  `last_position` INT DEFAULT 0 COMMENT '上次播放位置(秒)',
  `updated_at` DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
  UNIQUE KEY `uk_student_chapter` (`student_id`, `chapter_id`),
  FOREIGN KEY (`student_id`) REFERENCES `students`(`id`),
  FOREIGN KEY (`chapter_id`) REFERENCES `chapters`(`id`)
);

-- 题库表
CREATE TABLE `questions` (
  `id` VARCHAR(36) PRIMARY KEY,
  `category` VARCHAR(50) COMMENT '题目分类',
  `type` ENUM('single', 'multiple', '判断') NOT NULL COMMENT '题型',
  `content` TEXT NOT NULL COMMENT '题干',
  `options` JSON COMMENT '选项 [{"key":"A","value":"..."}]',
  `answer` VARCHAR(10) NOT NULL COMMENT '答案',
  `explanation` TEXT COMMENT '解析',
  `difficulty` TINYINT DEFAULT 1 COMMENT '难度1-5',
  `created_at` DATETIME DEFAULT CURRENT_TIMESTAMP
);

-- 试卷配置表
CREATE TABLE `exam_configs` (
  `id` VARCHAR(36) PRIMARY KEY,
  `name` VARCHAR(100) NOT NULL COMMENT '试卷名称',
  `duration` INT DEFAULT 60 COMMENT '考试时长(分钟)',
  `pass_score` INT DEFAULT 80 COMMENT '及格分数',
  `total_score` INT DEFAULT 100 COMMENT '总分',
  `question_count` INT DEFAULT 50 COMMENT '题目数量',
  `exam_times` INT DEFAULT 3 COMMENT '允许考试次数',
  `interval_hours` INT DEFAULT 24 COMMENT '重新考试间隔(小时)',
  `status` TINYINT DEFAULT 1,
  `created_at` DATETIME DEFAULT CURRENT_TIMESTAMP
);

-- 考试记录表
CREATE TABLE `exam_records` (
  `id` VARCHAR(36) PRIMARY KEY,
  `student_id` VARCHAR(36) NOT NULL COMMENT '学员ID',
  `exam_config_id` VARCHAR(36) NOT NULL COMMENT '试卷配置ID',
  `score` INT COMMENT '考试成绩',
  `passed` TINYINT COMMENT '是否及格',
  `answers` JSON COMMENT '答题详情',
  `correct_count` INT COMMENT '正确题数',
  `exam_time` DATETIME COMMENT '考试时间',
  `status` TINYINT DEFAULT 1 COMMENT '状态:1进行中2已完成',
  FOREIGN KEY (`student_id`) REFERENCES `students`(`id`),
  FOREIGN KEY (`exam_config_id`) REFERENCES `exam_configs`(`id`)
);

-- 证书表
CREATE TABLE `certificates` (
  `id` VARCHAR(36) PRIMARY KEY,
  `student_id` VARCHAR(36) NOT NULL COMMENT '学员ID',
  `cert_no` VARCHAR(50) NOT NULL UNIQUE COMMENT '证书编号',
  `project_name` VARCHAR(200) NOT NULL COMMENT '培训项目名称',
  `issue_date` DATE NOT NULL COMMENT '颁发日期',
  `expiry_date` DATE COMMENT '有效期',
  `template_id` VARCHAR(36) COMMENT '证书模板ID',
  `qr_code` VARCHAR(500) COMMENT '防伪二维码URL',
  `status` TINYINT DEFAULT 1 COMMENT '状态',
  `created_at` DATETIME DEFAULT CURRENT_TIMESTAMP,
  FOREIGN KEY (`student_id`) REFERENCES `students`(`id`)
);

-- 证书模板表
CREATE TABLE `cert_templates` (
  `id` VARCHAR(36) PRIMARY KEY,
  `name` VARCHAR(100) NOT NULL COMMENT '模板名称',
  `background` VARCHAR(500) COMMENT '背景图URL',
  `config` JSON COMMENT '模板配置(位置、字体等)',
  `created_at` DATETIME DEFAULT CURRENT_TIMESTAMP
);
```
