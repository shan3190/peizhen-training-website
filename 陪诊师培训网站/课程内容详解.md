> 本课程体系采用课本式教材编写风格，内容专业、系统、实用，覆盖从入门到精通的完整知识体系。

## 课程一：陪诊行业概述与职业规范

**课程类型：** 视频课程 + 图文教材  
**课程时长：** 45分钟  
**章节数量：** 5章  

### 课程简介

本课程作为陪诊师培训体系的入门课程，系统介绍陪诊行业的发展历程、市场现状与未来趋势，帮助学员建立正确的职业认知。课程重点讲解陪诊师的职业道德规范、职业素养要求、法律法规边界与职业风险防范，为后续专业技能学习奠定坚实的理论基础。

### 课本式核心内容摘要

#### 第一章 陪诊行业发展概述

**第一节 陪诊服务的定义与内涵**

陪诊服务是指专业陪诊人员陪同患者或其家属前往医疗机构就诊，提供全程导诊、代办手续、沟通协调、情感支持等专业化服务的新型医疗辅助服务模式。陪诊服务的核心价值在于：优化就医体验、提高就诊效率、缓解医患信息不对称、减轻家属陪护负担。

**第二节 陪诊行业发展历程**

我国陪诊行业起步于2015年前后，最初以零散的个体服务形式存在。2020年新冠疫情后，医疗资源紧张与老龄化加速推动行业快速发展。据《2024年中国陪诊行业白皮书》统计，全国陪诊从业人员已超30万人，市场规模突破200亿元。行业发展呈现三个阶段：
- 萌芽期（2015-2019）：个体化服务，缺乏规范
- 成长期（2020-2023）：平台化运作，规模扩张
- 规范期（2024-至今）：标准建立，品质升级

**第三节 陪诊服务市场需求分析**

市场需求主要来源于四大群体：
1. **老年患者群体**：60岁以上人口占比超19%，独居老人就医困难
2. **异地就医群体**：三甲医院异地就诊比例达40%，人生地不熟
3. **行动不便群体**：残障人士、术后康复患者等
4. **时间价值敏感群体**：商务人士、高收入家庭等

#### 第二章 陪诊师职业定位与素养

**第一节 陪诊师的角色定位**

陪诊师是医疗服务体系的重要补充力量，承担着多重角色：
- **就医导航员**：熟悉医院布局与流程，高效引导就诊
- **沟通协调员**：医患之间的桥梁，促进有效沟通
- **情感支持者**：缓解患者焦虑，提供心理陪伴
- **事务代办人**：代办挂号、缴费、取药等事务

**第二节 核心职业素养要求**

1. **职业道德素养**：诚实守信、尊重隐私、爱岗敬业、客观中立
2. **沟通表达素养**：倾听能力、表达能力、同理心、情绪管理
3. **专业知识素养**：医疗常识、医院流程、法规知识
4. **应急处理素养**：冷静判断、快速反应、安全意识

#### 第三章 职业道德与行为规范

**第一节 职业道德准则**

1. **尊重原则**：尊重患者人格、价值观、宗教信仰
2. **保密原则**：严格保护患者隐私信息，不泄露病情
3. **公正原则**：公平对待每一位患者，不差别对待
4. **有利原则**：始终以患者利益最大化为出发点
5. **诚信原则**：不虚夸服务、不隐瞒信息、不诱导消费

**第二节 行为规范细则**

1. **仪容仪表规范**：着装整洁、仪态端庄、标识清晰
2. **服务语言规范**：使用礼貌用语、避免专业术语、语速适中
3. **行为举止规范**：主动搀扶、适时提醒、保持距离
4. **纪律规范**：不迟到早退、不擅离职守、不收受红包

#### 第四章 法律法规与职业边界

**第一节 相关法律法规**

1. **《中华人民共和国民法典》**：合同关系、侵权责任、隐私权保护
2. **《基本医疗卫生与健康促进法》**：医疗服务规范
3. **《个人信息保护法》**：患者信息保护
4. **《医师法》**：明确陪诊师不得从事诊疗活动

**第二节 职业边界与红线**

陪诊师必须明确"三不原则"：
1. **不提供医疗建议**：不得解释检查报告、推荐治疗方案
2. **不代替医生决策**：不得代替患者或家属做出医疗决策
3. **不从事诊疗活动**：不得进行任何医疗操作

#### 第五章 职业风险与防范

**第一节 常见职业风险类型**

1. **法律风险**：合同纠纷、隐私泄露、医疗事故连带责任
2. **安全风险**：院内感染、意外伤害、突发疾病
3. **职业倦怠**：情绪耗竭、同理心疲劳、工作压力

**第二节 风险防范措施**

1. **合同防范**：签订规范服务协议，明确权责边界
2. **保险防范**：购买职业责任险、意外险
3. **流程防范**：标准化作业流程，记录留痕
4. **自我保护**：佩戴防护用品，定期健康检查

## 课程二：医院就诊流程全解析

**课程类型：** 视频课程 + 实操演练  
**课程时长：** 60分钟  
**章节数量：** 6章  

### 课程简介

本课程系统解析国内三甲医院的完整就诊流程，从预约挂号到取药结算的每个环节进行精细化讲解。通过大量真实案例和实操演练，学员将掌握不同类型医院的就诊特点、各科室分布规律、智能设备使用技巧，成为真正的"医院通"。

### 课本式核心内容摘要

#### 第一章 医院体系与科室设置

**第一节 我国医院分级体系**

根据《医院分级管理标准》，我国医院分为三级十等：
- **一级医院**：社区医院、乡镇卫生院，提供基础医疗服务
- **二级医院**：区县医院，提供综合医疗服务
- **三级医院**：省市三甲医院，提供疑难重症诊疗服务

**第二节 医院科室设置规律**

医院科室按功能分为四大类：
1. **门诊科室**：内科、外科、妇科、儿科、专科门诊
2. **医技科室**：检验科、影像科、超声科、病理科
3. **住院科室**：各专科病房、ICU、手术室
4. **辅助科室**：药房、收费处、导诊台、后勤保障

**第三节 三甲医院布局特点**

- **门诊楼**：通常位于医院入口处，一楼设挂号收费、药房、急诊科
- **医技楼**：紧邻门诊楼，设检验、影像等检查科室
- **住院楼**：相对独立区域，设各病区病房
- **公共区域**：食堂、便利店、停车场等配套设施

#### 第二章 预约挂号全攻略

**第一节 挂号渠道对比分析**

| 挂号渠道 | 优势 | 劣势 | 适用场景 |
|---------|------|------|---------|
| 官方APP/公众号 | 号源最全、可查报告 | 需要注册绑定 | 日常预约 |
| 114电话预约 | 操作简单、适合老人 | 号源有限 | 紧急预约 |
| 自助机挂号 | 现场挂号、即时可取 | 需要排队 | 当日就诊 |
| 窗口挂号 | 可咨询人工 | 排队时间长 | 特殊情况 |
| 第三方平台 | 多医院统一管理 | 号源延迟 | 多院对比 |

**第二节 号源释放规律**

1. **放号时间**：多数医院7:00-8:00放号，部分零点放号
2. **预约周期**：一般提前7天预约，热门科室可能14天
3. **退号捡漏**：就诊前1-2天是退号高峰期，重点关注10:00、14:00、16:00
4. **加号策略**：知名专家门诊结束前可尝试诊室加号

**第三节 挂号常见问题处理**

1. **号源已满**：查看同科室其他医生、选择特需门诊、考虑分院区
2. **挂错科室**：及时退号重挂，或到分诊台咨询改号
3. **无法按时就诊**：提前24小时取消，避免影响信用

#### 第三章 门诊就诊流程详解

**第一节 就诊前准备**

1. **资料准备**：身份证/医保卡、既往病历、检查报告、用药清单
2. **身体准备**：空腹检查注意禁食禁水、憋尿项目提前准备
3. **物品准备**：充电宝、水杯、纸巾、备用衣物
4. **信息准备**：整理症状时间线、准备问题清单

**第二节 门诊标准流程**

```
到达医院 → 取号/签到 → 候诊 → 就诊 → 开具检查/处方 → 缴费 → 检查/取药 → 复诊 → 离院
```

**关键环节详解：**

1. **签到环节**：签到后才进入叫号序列，提前30分钟签到
2. **候诊环节**：注意听叫号，过号需重新签到
3. **就诊环节**：简明扼要描述症状，准确回答医生问题
4. **缴费环节**：支持线上缴费、自助机、人工窗口

#### 第四章 检查检验流程指南

**第一节 常见检查类型与注意事项**

1. **血液检查**
   - 空腹项目：血糖、血脂、肝功能等，需禁食8-12小时
   - 非空腹项目：血常规、凝血功能等，可随时检查
   - 取报告时间：常规2-4小时，特殊项目1-3天

2. **影像检查**
   - X线：去除金属物品，配合呼吸指令
   - CT：部分需要增强造影，提前签署知情同意书
   - MRI：严禁携带金属物品，检查时间较长（15-30分钟）

3. **超声检查**
   - 腹部超声：空腹8小时
   - 妇科/前列腺超声：憋尿至有明显尿意
   - 心脏超声：无需特殊准备

**第二节 检查预约与加急策略**

1. **预约方式**：诊室预约、窗口预约、线上预约
2. **预约周期**：普通检查1-3天，特殊检查1-2周
3. **加急途径**：急诊科、病情危重证明、医生协调
4. **取消改期**：提前24小时操作，避免违约记录

#### 第五章 取药与结算流程

**第一节 药房取药流程**

1. **处方流转**：医生开具处方后自动传送到药房
2. **缴费确认**：线上或窗口缴费后处方生效
3. **药房签到**：到对应药房扫码签到
4. **等待叫号**：留意取药窗口叫号
5. **核对药品**：核对药名、数量、用法用量

**第二节 医保结算政策**

1. **医保类型**：职工医保、居民医保、新农合
2. **报销比例**：门诊50%-70%，住院70%-90%
3. **起付标准**：各级医院起付线不同
4. **异地就医**：需提前备案，执行"就医地目录、参保地政策"

**第三节 费用查询与票据**

1. **费用查询**：自助机、APP、收费窗口可打印费用明细
2. **票据获取**：收费窗口打印正式发票，注意保存
3. **退费流程**：未执行项目可申请退费，需医生签字确认

#### 第六章 特殊场景就诊处理

**第一节 急诊就诊流程**

急诊就诊优先级：
1. **一级（急危症）**：心跳骤停、休克等，立即抢救
2. **二级（急重症）**：胸痛、呼吸困难等，10分钟内处理
3. **三级（急症）**：高热、急性腹痛等，30分钟内处理
4. **四级（非急症）**：普通急诊，按顺序就诊

**第二节 住院办理流程**

1. **开具住院证**：门诊医生评估后开具
2. **缴纳押金**：按病种预交，一般3000-10000元
3. **办理入院**：住院处登记，领取腕带
4. **病区报到**：到对应护士站办理入科
5. **完善检查**：入院常规检查，等待手术/治疗安排

## 课程三：常见疾病知识与沟通技巧

**课程类型：** 视频课程 + 案例分析  
**课程时长：** 75分钟  
**章节数量：** 7章  

### 课程简介

本课程精选临床最常见的20余种疾病进行系统讲解，包括症状识别、就诊科室、检查项目、治疗原则等核心知识。同时深入探讨医患沟通技巧，学习如何与不同性格、不同病情的患者有效沟通，提升服务专业性和患者满意度。

### 课本式核心内容摘要

#### 第一章 常见疾病知识学习方法论

**第一节 陪诊师医学知识定位**

陪诊师掌握医学知识的"三度原则"：
1. **广度**：了解常见疾病基本常识，覆盖面要广
2. **深度**：不深入研究诊疗方案，避免越界
3. **准确度**：准确理解医学术语，不传播错误信息

**第二节 医学知识学习框架**

每一种疾病需掌握"五要素"：
1. **识别要点**：典型症状、预警信号
2. **就诊科室**：首诊科室、专科细分
3. **检查项目**：必查项目、辅助检查
4. **治疗原则**：治疗方向、不涉及具体方案
5. **注意事项**：就诊准备、生活护理

#### 第二章 循环系统常见疾病

**第一节 高血压病**

**识别要点：**
- 典型症状：头晕、头痛、颈项板紧、疲劳、心悸
- 血压标准：收缩压≥140mmHg和/或舒张压≥90mmHg
- 危险信号：剧烈头痛、呕吐、意识障碍（高血压急症）

**就诊科室：** 心血管内科  
**必查项目：** 血压测量、血常规、尿常规、肝肾功能、心电图  
**治疗原则：** 生活方式干预 + 药物治疗，长期控制  
**陪诊注意事项：** 提醒按时服药、监测血压、低盐饮食

**第二节 冠心病**

**识别要点：**
- 典型心绞痛：胸骨后压榨感、紧缩感，向左肩放射
- 诱因：劳累、情绪激动、饱餐、寒冷
- 持续时间：3-5分钟，休息或含服硝酸甘油缓解
- 心梗预警：疼痛加剧、持续时间延长、休息不缓解

**就诊科室：** 心血管内科（急诊）  
**必查项目：** 心电图、心肌酶谱、冠脉CTA/造影  
**治疗原则：** 药物治疗、介入治疗、手术治疗  
**陪诊注意事项：** 随身携带硝酸甘油、避免剧烈活动

#### 第三章 呼吸系统常见疾病

**第一节 急性上呼吸道感染**

**识别要点：**
- 普通感冒：鼻塞、流涕、咽痛、咳嗽、低热
- 流行性感冒：高热、全身酸痛、乏力明显
- 细菌性感染：黄脓痰、扁桃体化脓、白细胞升高

**就诊科室：** 呼吸内科/全科医学科  
**必查项目：** 血常规、C反应蛋白  
**治疗原则：** 对症支持、抗病毒（流感）、抗菌（细菌）  
**陪诊注意事项：** 多休息、多饮水、戴口罩防护

**第二节 肺炎**

**识别要点：**
- 典型症状：发热、咳嗽、咳痰、胸痛、呼吸困难
- 重症信号：呼吸急促（>30次/分）、血氧降低、意识模糊
- 特殊人群：老年人可能仅表现为精神萎靡、食欲下降

**就诊科室：** 呼吸内科（重症需急诊）  
**必查项目：** 胸片/CT、血常规、血气分析  
**治疗原则：** 抗感染、对症支持、氧疗  
**陪诊注意事项：** 监测体温、观察呼吸状态、协助排痰

#### 第四章 消化系统常见疾病

**第一节 慢性胃炎/消化性溃疡**

**识别要点：**
- 胃炎：上腹痛、腹胀、嗳气、反酸
- 胃溃疡：餐后痛，进食后加重
- 十二指肠溃疡：饥饿痛、夜间痛，进食后缓解
- 危险信号：黑便、呕血（消化道出血）

**就诊科室：** 消化内科  
**必查项目：** 胃镜、幽门螺杆菌检测  
**治疗原则：** 抑酸、保护胃黏膜、根除幽门螺杆菌  
**陪诊注意事项：** 规律饮食、避免辛辣刺激、分餐（防传染）

**第二节 急性阑尾炎**

**识别要点：**
- 典型表现：转移性右下腹痛（始于上腹痛→右下腹）
- 伴随症状：恶心、呕吐、发热
- 体征：右下腹压痛、反跳痛、肌紧张

**就诊科室：** 外科急诊  
**必查项目：** 血常规、腹部超声/CT  
**治疗原则：** 手术治疗为主，单纯性可保守  
**陪诊注意事项：** 禁食水、避免止痛药掩盖病情

#### 第五章 内分泌与代谢疾病

**第一节 糖尿病**

**识别要点：**
- 典型"三多一少"：多饮、多食、多尿、体重减轻
- 不典型表现：皮肤瘙痒、反复感染、伤口不愈合
- 诊断标准：空腹血糖≥7.0mmol/L或餐后2小时≥11.1mmol/L

**就诊科室：** 内分泌科  
**必查项目：** 空腹血糖、糖化血红蛋白、尿常规  
**治疗原则：** 五驾马车（教育、饮食、运动、药物、监测）  
**陪诊注意事项：** 携带糖果预防低血糖、定时监测血糖

**第二节 甲状腺疾病**

**识别要点：**
- 甲亢：心慌、手抖、多汗、体重下降、易怒
- 甲减：怕冷、乏力、体重增加、嗜睡、记忆力减退
- 甲状腺结节：颈部肿块，多数无症状

**就诊科室：** 内分泌科/甲乳外科  
**必查项目：** 甲状腺功能、甲状腺超声  
**治疗原则：** 药物治疗、手术治疗、碘131治疗  
**陪诊注意事项：** 定期复查、避免过多含碘食物（甲亢）

#### 第六章 医患沟通核心技巧

**第一节 沟通基础原则**

**SOLER沟通技巧：**
- **S(Squarely)**：面向患者，保持适当距离
- **O(Open)**：开放姿态，不抱臂交叉
- **L(Lean)**：适当前倾，表示关注
- **E(Eye)**：眼神接触，不躲闪不移开
- **R(Relax)**：放松状态，自然从容

**第二节 与不同类型患者沟通策略**

1. **焦虑型患者**
   - 特点：过度担心、反复询问、情绪不稳定
   - 策略：耐心倾听、信息确认、适度保证、提供事实

2. **沉默型患者**
   - 特点：少言寡语、不主动表达、顺从被动
   - 策略：开放式提问、耐心等待、非语言鼓励、细心观察

3. **要求型患者**
   - 特点：要求多、标准高、易不满意
   - 策略：明确边界、专业解答、适当共情、管理预期

4. **老年患者**
   - 特点：听力下降、记忆力减退、理解慢
   - 策略：语速放慢、重复重点、简化表达、书面提醒

#### 第七章 病情告知与坏消息传递

**第一节 病情告知基本原则**

1. **尊重原则**：尊重患者知情权，不隐瞒重要信息
2. **渐进原则**：逐步告知，给患者心理缓冲时间
3. **个体化原则**：根据患者心理承受能力调整方式
4. **陪伴原则**：告知后陪伴患者，提供情感支持

**第二节 坏消息传递六步法（SPIKES）**

1. **S(Settings)**：准备环境，确保隐私
2. **P(Perception)**：了解患者已知信息
3. **I(Invitation)**：询问患者是否愿意知晓
4. **K(Knowledge)**：传递信息，简洁明确
5. **E(Emotions)**：回应患者情绪反应
6. **S(Strategy/Summary)**：总结下一步计划

## 课程四：老年人陪护与心理疏导

**课程类型：** 视频课程 + 情景模拟  
**课程时长：** 70分钟  
**章节数量：** 6章  

### 课程简介

老年人是陪诊服务的核心客户群体，其生理特点和心理需求具有显著特殊性。本课程深入讲解老年人生理机能衰退特点、常见老年疾病陪护要点、老年心理特征与沟通技巧、认知障碍照护方法等专业内容，培养专业的老年陪护能力。

### 课本式核心内容摘要

#### 第一章 老年人身心特点概述

**第一节 生理机能衰退特点**

老年人身体系统退行性变化：

1. **感官系统**
   - 视觉：老花眼、白内障、对比敏感度下降
   - 听觉：高频听力下降、耳鸣、语言辨识困难
   - 味觉：味觉减退，口味偏重

2. **运动系统**
   - 肌肉萎缩、肌力下降、易疲劳
   - 关节退行性变，活动受限
   - 平衡能力下降，跌倒风险高

3. **神经系统**
   - 反应速度减慢
   - 短期记忆力下降
   - 认知处理能力降低

**第二节 心理特征变化**

1. **认知变化**：液态智力下降，晶态智力保持
2. **情绪变化**：易焦虑、抑郁、孤独感增强
3. **性格变化**：固执、怀旧、自我中心倾向
4. **需求变化**：安全感需求、归属感需求、尊重需求

#### 第二章 老年常见疾病陪护要点

**第一节 心血管疾病陪护**

**高血压老年患者：**
- **监测要点**：固定时间、固定体位、固定血压计测量
- **用药提醒**：按时服药，不随意增减剂量
- **活动指导**：避免突然起身（防体位性低血压）
- **饮食建议**：低盐低脂，每日盐摄入<5g

**冠心病老年患者：**
- **活动管理**：循序渐进，以不感疲劳为度
- **急救准备**：随身携带急救药物
- **情绪管理**：避免激动、生气、过度兴奋
- **症状识别**：学会识别不典型心绞痛（牙痛、肩痛）

**第二节 糖尿病老年患者陪护**

**日常陪护要点：**
1. **血糖监测**：空腹+三餐后2小时，记录数据
2. **饮食管理**：定时定量，粗粮搭配，避免高糖
3. **运动指导**：餐后1小时运动，每次30分钟
4. **足部护理**：每日检查，温水洗脚，避免烫伤
5. **低血糖处理**：立即口服糖水或糖果，15分钟复测

**第三节 骨关节疾病陪护**

**骨质疏松/骨关节炎陪护：**
- **防跌倒**：清除路障，改善照明，辅助行走
- **补钙指导**：牛奶、豆制品、钙剂+维生素D
- **疼痛管理**：热敷、理疗、按医嘱用药
- **康复锻炼**：关节活动度训练、肌力训练

#### 第三章 老年人沟通专项技巧

**第一节 语言沟通技巧**

**"三慢三多"原则：**
- **语速慢**：语速放缓，每分钟约100字
- **反应慢**：提问后等待3-5秒再重复
- **动作慢**：转换话题、变换动作都要慢
- **多重复**：重要信息重复3遍
- **多确认**：说完后让老人复述确认
- **多鼓励**：及时肯定，增强沟通信心

**沟通注意事项：**
1. **说话位置**：面对老人，视线平齐，距离1米内
2. **音量控制**：稍大但不叫喊，低音调比高音调更容易听清
3. **用词简单**：避免专业术语和网络用语
4. **表达具体**：不说"过一会儿"，说"等10分钟"

**第二节 非语言沟通技巧**

1. **面部表情**：微笑、温和、专注
2. **肢体接触**：握手、拍肩、搀扶（先征得同意）
3. **肢体语言**：点头、前倾表示关注
4. **环境配合**：安静环境、充足光线、舒适座椅

#### 第四章 老年心理疏导实务

**第一节 常见心理问题识别**

1. **焦虑情绪识别**
   - 表现：坐立不安、反复询问、失眠、食欲下降
   - 诱因：对疾病担忧、对未知恐惧、经济压力
   - 评估：焦虑自评量表（SAS），>50分提示焦虑

2. **抑郁情绪识别**
   - 表现：情绪低落、兴趣减退、自责、自杀念头
   - 警示："活着没意思"、"还不如死了好"等语言
   - 评估：抑郁自评量表（SDS），>53分提示抑郁

3. **孤独感识别**
   - 表现：沉默寡言、社交回避、过度依赖陪诊人员
   - 对策：耐心陪伴、倾听倾诉、鼓励社交

**第二节 心理疏导基本方法**

**共情三步法：**
1. **识别情绪**："我理解您现在很担心..."
2. **接纳情绪**："有这种担心是很正常的..."
3. **正常化情绪**："很多人遇到这种情况都会这样..."

**认知调整技术：**
1. **灾难化思维矫正**："最坏的结果是什么？发生概率多大？"
2. **非黑即白思维调整**："除了A和B，还有没有其他可能？"
3. **过度概括纠正**："这一次不顺利，不等于以后都不顺利"

#### 第五章 认知障碍患者照护

**第一节 认知障碍分期与表现**

阿尔茨海默病分期：

1. **轻度认知障碍期（1-3年）**
   - 表现：近期记忆力下降、学习困难、判断力轻度受损
   - 陪护：提醒记忆、建立规律、环境熟悉

2. **中度痴呆期（2-8年）**
   - 表现：远期记忆也受损、定向障碍、语言障碍、人格改变
   - 陪护：全程陪同、防止走失、生活协助

3. **重度痴呆期（8-12年）**
   - 表现：完全依赖照护、大小便失禁、缄默
   - 陪护：基础护理、营养支持、预防并发症

**第二节 认知障碍沟通技巧**

**沟通十原则：**
1. **环境安静**：减少干扰，一对一沟通
2. **吸引注意**：呼唤名字，眼神接触
3. **语句简短**：每句话只说一件事
4. **语速放慢**：给大脑足够处理时间
5. **避免提问**：不说"您记得吗？"改为"我们一起..."
6. **多做示范**：用动作演示代替语言
7. **不纠正错误**：避免争论，顺着说
8. **反复重复**：同一信息多次重复
9. **多用肯定**："对"、"好的"、"没问题"
10. **保持耐心**：不催促、不急躁、不放弃

#### 第六章 老年人安全陪护

**第一节 跌倒预防与处理**

**跌倒风险评估**：
- 年龄≥75岁
- 既往跌倒史
- 步态不稳
- 服用降压/镇静/降糖药
- 视力障碍

**预防措施**：
1. **行走辅助**：适当使用助行器（手杖/助步器）
2. **穿鞋合适**：防滑软底鞋，不穿拖鞋
3. **环境安全**：清除地面积水、障碍物
4. **起身缓慢**：起床"三部曲"：醒30秒→坐30秒→站30秒

**跌倒后处理**：
1. 不要急于扶起，先评估意识和伤情
2. 询问哪里痛，观察肢体活动
3. 怀疑骨折时不要移动，立即呼叫医护
4. 无明显损伤时协助缓慢起身，继续观察

## 课程五：特殊患者陪护技能

**课程类型：** 视频课程 + 专项训练  
**课程时长：** 65分钟  
**章节数量：** 5章  

### 课程简介

本课程聚焦特殊群体的专业陪护技能，包括儿科患者陪护、残障人士陪护、重症患者陪护、传染病防护等专项内容。通过系统学习和专项训练，学员将掌握各类特殊患者的陪护要点、沟通技巧和注意事项，满足多元化的服务需求。

### 课本式核心内容摘要

#### 第一章 儿科患者陪护技能

**第一节 儿科就诊特点**

儿科"哑科"特点：
- 患儿不会准确描述症状
- 家长焦虑程度高
- 病情变化快
- 配合度低，检查困难

**不同年龄段特点：**

1. **婴儿期（0-1岁）**
   - 特点：完全依赖、恐惧陌生人、易哭闹
   - 陪护：保持舒适、轻声安抚、分散注意力

2. **幼儿期（1-3岁）**
   - 特点：自我意识强、抗拒检查、分离焦虑
   - 陪护：游戏化引导、夸赞鼓励、家长陪伴

3. **学龄前期（3-6岁）**
   - 特点：好奇心强、想象力丰富、怕打针
   - 陪护：故事引导、提前预告、榜样示范

4. **学龄期（6-12岁）**
   - 特点：懂道理、自尊心强、怕疼但能配合
   - 陪护：尊重理解、解释说明、给予选择权

**第二节 儿科陪护沟通技巧**

**与患儿沟通"四步法"：**
1. **建立关系**：蹲下来、微笑、叫名字、适当夸奖
2. **解释说明**：用孩子能懂的语言，比喻说明
3. **参与互动**：让孩子做简单选择，参与部分操作
4. **及时奖励**：配合后给予表扬或小奖品

**安抚哭闹技巧：**
- 怀抱安抚：竖抱、轻拍、温柔说话
- 感官分散：看卡通、玩玩具、听儿歌
- 吸吮安抚：奶嘴、奶瓶、小零食
- 环境转移：换房间、看窗外、看其他小朋友

**第三节 常见儿科检查配合技巧**

1. **采血配合**
   - 选择合适部位，避开瘢痕
   - 家长正确固定姿势（搂抱+按压）
   - 操作中持续安抚，操作后及时表扬
   - 按压5-10分钟，避免揉搓

2. **影像检查配合**
   - 提前沟通，用游戏方式解释
   - 家长陪同进入检查室
   - 必要时使用镇静剂（遵医嘱）

#### 第二章 残障人士陪护技能

**第一节 视力障碍陪护**

**服务原则：**
- 看见即告知：进出房间、有人进出都要告知
- 听到即回应：呼唤时立即应答，让对方知道位置
- 离开需说明：暂时离开要说明去向和返回时间

**引导行走技巧：**
1. 让视障者扶着陪护者的上臂（不是搀扶）
2. 陪护者走在前半步，随步伐自然摆动
3. 遇台阶/门槛提前提醒："前面有台阶，上3阶"
4. 落座引导：将手放在椅背上，让对方感知位置

**沟通注意事项：**
- 说话前先称呼名字，让对方知道在对他/她说话
- 正常说话，不需要提高音量
- 避免用"你看"、"你瞧"等视觉词汇
- 进出时告知，让对方有心理准备

**第二节 听力障碍陪护**

**沟通方式选择：**
1. **口语沟通**：面对面、语速慢、口型清晰、配合手势
2. **笔谈沟通**：准备纸笔或手机文字输入
3. **手语沟通**：学习基础手语，借助翻译软件
4. **辅助工具**：助听器、人工耳蜗注意电量和佩戴

**书写沟通技巧：**
- 字体工整，字迹清晰
- 内容简短，每次只写一个问题
- 使用大白话，避免专业术语
- 重要信息让对方复述确认

**第三节 肢体障碍陪护**

**移动辅助技巧：**
1. **轮椅使用**
   - 上下坡道：正面推上坡，倒推下坡
   - 上下台阶：需要两人配合，轮椅后倾抬行
   - 乘坐电梯：先退后进，面朝电梯门
   - 长时间停留：踩下刹车锁轮

2. **转移辅助**
   - 轮椅↔座椅：先刹轮椅，靠近座椅，辅助站立转向
   - 轮椅↔床：侧对床边，刹轮椅，辅助移动
   - 注意事项：站在患者弱侧，保护重心，避免牵拉

**生活协助注意事项：**
- 尊重自主，只在需要时帮助，不要过度协助
- 协助前询问需求方式："需要我帮您吗？"
- 保护隐私，如厕、更衣等私密活动尽量让患者自主完成

#### 第三章 重症患者陪护技能

**第一节 ICU探视陪护**

**ICU环境特点：**
- 仪器设备多，报警声频繁
- 患者多无意识或意识模糊
- 探视时间有限，管理严格
- 家属焦虑程度高，情绪易激动

**陪护要点：**
1. **探视准备**
   - 按规定更衣、换鞋、戴口罩
   - 患有感冒等传染病时不探视
   - 手机静音，不拍摄患者照片

2. **与重症患者沟通**
   - 即使无意识也要说话，听力可能尚存
   - 声音温和，告知"我来看你了"
   - 轻轻触碰（征得医护同意），传递温暖
   - 说熟悉的人和事，帮助唤醒

3. **家属支持**
   - 倾听家属倾诉，接纳情绪
   - 解释ICU规则和治疗流程
   - 协助与医护沟通，了解病情
   - 提醒休息，避免过度疲劳

**第二节 术后患者陪护**

**术后早期陪护要点：**
1. **生命体征观察**：意识、呼吸、血压、体温
2. **伤口护理**：观察渗血、保持清洁、避免牵拉
3. **管道护理**：输液管、引流管、氧气管妥善固定，避免脱出
4. **疼痛管理**：评估疼痛程度，按医嘱使用止痛药
5. **早期活动**：床上翻身→坐起→床边站立→行走（遵医嘱）

**术后常见不适处理：**
- **恶心呕吐**：头偏向一侧，防止误吸，告知护士
- **腹胀**：早期下床活动，腹部按摩（遵医嘱）
- **排尿困难**：诱导排尿（听流水声），必要时导尿
- **睡眠障碍**：创造安静环境，减少日间睡眠

#### 第四章 传染病防护与陪护

**第一节 常见传染病类型与防护等级**

**传染病传播途径：**
- **呼吸道传播**：流感、结核、新冠等（空气/飞沫）
- **消化道传播**：甲肝、痢疾、诺如病毒等（粪口）
- **接触传播**：手足口病、皮肤感染等（直接/间接）
- **血液体液传播**：乙肝、丙肝、艾滋病等（血液/体液）

**防护分级：**

| 防护级别 | 适用场景 | 防护装备 |
|---------|---------|---------|
| 一级防护 | 普通门诊、常规陪护 | 医用外科口罩、手卫生 |
| 二级防护 | 发热门诊、疑似病例 | N95口罩、护目镜、隔离衣、手套 |
| 三级防护 | 确诊病例、有创操作 | 全套装备+全面型呼吸防护器 |

**第二节 标准预防措施**

**手卫生"两前三后"：**
- 接触患者前
- 无菌操作前
- 接触患者后
- 接触患者周围环境后
- 接触血液体液后

**七步洗手法：** 内→外→夹→弓→大→立→腕（每步15秒）

**个人防护装备使用：**
1. **口罩佩戴**：完全覆盖口鼻，压实鼻夹，不漏气
2. **手套使用**：接触血液体液时戴，及时更换
3. **隔离衣**：可能被污染时穿，一次性使用
4. **护目镜**：有喷溅风险时使用

#### 第五章 临终关怀与舒缓陪护

**第一节 临终患者心理变化**

库伯勒-罗斯临终五阶段：
1. **否认期**："不可能是我"，拒绝接受现实
2. **愤怒期**："为什么是我"，愤怒抱怨，迁怒他人
3. **协议期**："如果让我好起来，我就..."，讨价还价
4. **抑郁期**：悲伤、哭泣、沉默，安排后事
5. **接受期**：平静、安详、准备告别

**陪护原则：**
1. 接纳任何情绪表现，不评判、不说教
2. 陪伴倾听，比说什么更重要
3. 尊重患者意愿，包括放弃治疗的选择
4. 帮助完成未了心愿，减少遗憾

**第二节 临终关怀实务技巧**

**舒适护理：**
- **疼痛控制**：按阶梯用药，维持无痛状态
- **口腔护理**：保持湿润，预防感染
- **皮肤护理**：定时翻身，预防压疮
- **环境舒适**：温度适宜、光线柔和、噪音最小

**灵性关怀：**
- 回顾人生，肯定价值
- 满足宗教信仰需求
- 促进家庭和解
- 帮助与亲友道别

**家属支持：**
- 允许家属陪伴最后时光
- 指导家属如何陪伴
- 提供哀伤辅导资源
- 帮助安排后事

## 课程六：陪诊服务运营与客户管理

**课程类型：** 视频课程 + 实操训练  
**课程时长：** 55分钟  
**章节数量：** 5章  

### 课程简介

本课程聚焦陪诊服务的商业化运营，包括服务流程标准化、客户开发与维护、服务定价与套餐设计、投诉处理与品质提升等核心内容。帮助学员不仅具备专业技能，更掌握运营能力，实现从"专业陪诊师"到"成功创业者"的跨越。

### 课本式核心内容摘要

#### 第一章 陪诊服务标准化流程

**第一节 服务全流程标准化**

标准服务流程SOP：

```
接单确认 → 服务准备 → 院前对接 → 院内陪诊 → 服务收尾 → 售后跟进
```

**各环节标准动作：**

1. **接单确认（5分钟内）**
   - 联系客户，确认时间地点
   - 了解患者基本情况（年龄、病情、行动能力）
   - 明确服务内容与特殊需求
   - 发送服务须知与注意事项

2. **服务准备（提前1天）**
   - 预约挂号确认
   - 准备服务物料（工作牌、饮用水、纸巾、充电宝）
   - 查询医院科室位置
   - 规划交通路线，计算路程时间

3. **院前对接（就诊当日）**
   - 提前30分钟到达约定地点
   - 主动自我介绍，出示工作证件
   - 再次确认携带资料完整性
   - 简单沟通当天流程安排

4. **院内陪诊（全程陪同）**
   - 取号签到→排队候诊→陪同就诊→缴费→检查→取药
   - 每完成一项主动告知下一步
   - 重要信息及时记录
   - 突发情况立即处理并反馈

5. **服务收尾（结束后）**
   - 核对当日所有票据
   - 整理检查报告与医嘱
   - 提醒复查与用药注意事项
   - 护送患者安全离开

6. **售后跟进（24小时内）**
   - 发送服务小结与注意事项
   - 询问服务满意度
   - 提醒后续预约事项
   - 收集改进建议

**第二节 服务记录标准化**

**服务日志必填项：**
- 客户基本信息（姓名、年龄、联系方式）
- 服务时间、地点、医院科室
- 服务内容与时长
- 就诊医生与诊断结论
- 检查项目与结果摘要
- 用药建议与注意事项
- 下次复查时间
- 客户反馈与特殊情况

#### 第二章 客户开发与维护

**第一节 目标客户画像分析**

**核心客户群体：**

1. **子女不在身边的独居老人**
   - 年龄：65-85岁
   - 特点：行动不便、子女在外、有一定支付能力
   - 获取渠道：社区、老年大学、养老机构

2. **异地就医患者及家属**
   - 年龄：不限
   - 特点：不熟悉本地医院、需要全程代办
   - 获取渠道：医院周边、线上平台、病友群

3. **产后/术后康复期女性**
   - 年龄：25-45岁
   - 特点：行动不便、需要悉心照料
   - 获取渠道：月子中心、康复机构、母婴社群

4. **高收入时间敏感人群**
   - 年龄：30-50岁
   - 特点：重视效率、愿意为时间付费
   - 获取渠道：高端社区、企业合作、私人推荐

**第二节 获客渠道与方法**

1. **线上渠道**
   - 微信朋友圈/视频号：真实案例、服务展示
   - 抖音/小红书：科普内容、服务vlog
   - 美团/大众点评：店铺入驻、用户评价
   - 闲鱼/58同城：服务发布、低价引流

2. **线下渠道**
   - 医院周边：导诊台合作、周边商户推荐
   - 社区合作：居委会、物业、社区卫生中心
   - 机构合作：体检中心、牙科诊所、药店
   - 口碑转介绍：老客户推荐返利机制

3. **内容获客**
   - 医院就诊攻略系列文章
   - 常见疾病科普短视频
   - 陪诊真实案例分享
   - 免费问诊咨询引流

**第三节 客户关系维护**

**客户分层管理：**

| 客户层级 | 定义 | 维护策略 |
|---------|------|---------|
| A类客户 | 高价值、高复购、转介绍多 | VIP专属服务、节日礼品、优先排期 |
| B类客户 | 普通客户、有复购可能 | 定期回访、服务提醒、活动通知 |
| C类客户 | 一次性客户 | 满意度调查、偶尔触达、争取转化 |

**维护要点：**
1. **建立客户档案**：记录每次服务详情、偏好特点
2. **定期回访机制**：服务后24小时、7天、30天回访
3. **节日关怀**：生日、重要节日发送祝福
4. **知识传递**：定期发送健康科普、就医攻略
5. **专属福利**：老客户优惠、积分兑换、推荐奖励

#### 第三章 服务定价与套餐设计

**第一节 定价策略与依据**

**定价影响因素：**
1. **地域因素**：一线城市vs二三线城市
2. **服务内容**：基础陪诊vs全程代办vs专业陪护
3. **时长因素**：按小时vs按半天/全天
4. **时段因素**：工作日vs周末vs夜间急诊
5. **人员资质**：普通陪诊vs医护背景陪诊

**参考定价标准（一线城市）：**
- 基础陪诊：150-200元/半天（4小时）
- 全程陪诊：250-350元/全天（8小时）
- 特需服务：300-500元/次（专家号、加急）
- 夜间急诊：300-400元/次（18:00-8:00）
- 异地就医：500-800元/天（含住宿补贴）

**定价方法：**
1. **成本加成法**：（人工+交通+物料）×利润率
2. **市场对比法**：参考竞争对手价格上下浮动
3. **价值定价法**：根据节省的时间、提供的价值定价
4. **差别定价法**：不同客户、不同时段不同价格

**第二节 套餐产品设计**

**标准化套餐示例：**

1. **单次基础陪诊套餐**
   - 服务时长：4小时
   - 服务内容：取号、候诊、陪同就诊、缴费、取药
   - 适用人群：普通门诊、行动方便患者
   - 定价：198元

2. **全程无忧陪诊套餐**
   - 服务时长：8小时
   - 服务内容：全程陪同、所有环节代办、午餐安排、护送回家
   - 适用人群：老年患者、行动不便、复杂检查
   - 定价：368元

3. **体检陪同套餐**
   - 服务时长：4-6小时
   - 服务内容：引导各科室、排队代办、领取报告、解读说明
   - 适用人群：体检客户
   - 定价：298元

4. **VIP尊享套餐（年卡）**
   - 服务次数：12次全年通用
   - 服务内容：优先排期、专属客服、家属陪同优惠
   - 适用人群：高端客户、慢性病患者
   - 定价：3680元

#### 第四章 服务品质与投诉处理

**第一节 服务质量控制点**

**关键时刻（MOT）管理：**

1. **第一印象**
   - 准时到达（提前30分钟）
   - 仪容仪表（着装整洁、佩戴工牌）
   - 自我介绍（微笑、握手、说明身份）
   - 流程说明（简单清晰说明当天安排）

2. **沟通互动**
   - 主动沟通（每一步都告知）
   - 耐心倾听（不打断、不急于解释）
   - 专业解答（知之为知之，不知为不知）
   - 情绪支持（患者焦虑时及时安抚）

3. **问题处理**
   - 突发情况（冷静应对、及时反馈）
   - 意外延误（提前告知、说明原因）
   - 客户不满（立即道歉、寻求解决）
   - 特殊需求（尽力满足、超出能力说明）

4. **服务收尾**
   - 资料整理（票据、报告整理好）
   - 注意事项（用药、复查提醒）
   - 护送离开（送到门口/车上）
   - 后续跟进（当天晚上回访）

**第二节 投诉处理流程与技巧**

**投诉处理五步法：**

1. **耐心倾听**
   - 让客户把话说完，不打断、不辩解
   - 认真记录投诉要点
   - 适当点头、"嗯"表示在听
   - 保持冷静，不被情绪带动

2. **真诚道歉**
   - 无论对错，先为不好的体验道歉
   - "对不起，给您带来不好的体验"
   - 不找借口、不推卸责任
   - 态度真诚，语言诚恳

3. **确认问题**
   - 复述确认客户投诉内容
   - "您是说...对吗？"
   - 明确客户期望的解决方案
   - 不确定时询问："您希望我们如何处理？"

4. **解决问题**
   - 能当场解决的当场解决
   - 不能当场解决的说明时限
   - 提供多种方案供选择
   - 适当补偿（优惠券、免费服务）

5. **感谢反馈**
   - 感谢客户提出宝贵意见
   - 说明改进措施
   - 后续跟进确认满意度
   - 内部复盘，避免再犯

**投诉处理禁忌：**
- 与客户争辩、指责客户
- 说"这不是我的错"、"公司规定就是这样"
- 拖延处理、不了了之
- 轻易承诺做不到的事情
- 过度补偿，造成后续麻烦

#### 第五章 团队管理与规模化发展

**第一节 陪诊师团队建设**

**人员招募标准：**
1. **基本条件**：25-50岁，身体健康，无犯罪记录
2. **能力要求**：沟通能力强、有服务意识、应变能力好
3. **优先条件**：医护背景、护理专业、有陪护经验

**培训体系：**
- 岗前培训（7天）：理论知识、流程规范、实操演练
- 在岗培训（3个月）：师傅带教、案例研讨、技能提升
- 定期培训（每月）：新知识、新政策、服务升级

**考核激励：**
- 服务时长考核
- 客户满意度评分
- 投诉率控制
- 转介绍率奖励
- 阶梯式提成机制

**第二节 规模化发展路径**

发展三阶段：
1. **个人接单期（0-6个月）**
   - 积累经验、建立口碑、打磨流程
   - 目标：月服务30单以上，客户满意度95%+

2. **小团队期（6-18个月）**
   - 招募3-5人，建立培训和管理体系
   - 目标：月服务100单以上，形成标准化SOP

3. **公司化运营（18个月以上）**
   - 注册公司，建立品牌，多渠道获客
   - 目标：团队20人以上，覆盖多家医院，形成区域影响力

**关键成功要素：**
- **专业能力**：医疗知识、医院熟悉度、服务技能
- **服务品质**：标准化流程、细节把控、客户体验
- **口碑传播**：客户满意度、转介绍率、复购率
- **运营效率**：流程优化、人员调度、成本控制
- **风险控制**：合同规范、保险保障、应急预案
