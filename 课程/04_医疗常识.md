# 课程4：医疗常识

---

## 【学习目标】
通过本课程学习，学员将掌握人体基本结构和生理常识，了解常见疾病的认知和预防，学会解读常见检查项目，掌握常用药物基本知识，具备化验单阅读入门能力，并能将这些知识运用到实际陪诊服务中。

## 第一章 人体基本结构与生理常识

### 1.1 人体基本结构概述

#### 1.1.1 人体组成层次

**化学水平**：
- 人体由60多种元素组成
- 主要元素：氧、碳、氢、氮（占96%）
- 其他元素：钙、磷、钾、硫、钠、氯、镁等

**细胞水平**：
- 细胞是人体结构和功能的基本单位
- 人体约有40万亿-60万亿个细胞
- 不同细胞有不同功能：肌细胞、神经细胞、血细胞等

**组织水平**：
- 形态相似、功能相同的细胞和细胞间质构成组织
- 四大基本组织：上皮组织、结缔组织、肌肉组织、神经组织

**器官水平**：
- 不同组织结合成具有一定形态和功能的器官
- 如心、肺、肝、肾、胃、脑等

**系统水平**：
- 若干器官组合成系统，共同完成某一生理功能
- 人体有九大系统

#### 1.1.2 人体九大系统

| 系统名称 | 主要器官 | 主要功能 |
|----------|----------|----------|
| 运动系统 | 骨、关节、肌肉 | 运动、支持、保护 |
| 消化系统 | 口腔、食管、胃、肠、肝、胰 | 消化食物、吸收营养 |
| 呼吸系统 | 鼻、咽、喉、气管、肺 | 气体交换、吸入氧气排出二氧化碳 |
| 泌尿系统 | 肾、输尿管、膀胱、尿道 | 排出代谢废物和多余水分 |
| 生殖系统 | 男女内外生殖器 | 繁衍后代 |
| 内分泌系统 | 垂体、甲状腺、胰岛、肾上腺 | 调节代谢、生长发育等生理功能 |
| 免疫系统 | 淋巴结、脾、胸腺、骨髓 | 防御病原体、清除异己 |
| 神经系统 | 脑、脊髓、神经 | 调节和控制全身各系统活动 |
| 循环系统 | 心脏、血管、血液 | 运输营养和氧气，带走废物 |

#### 1.1.3 人体重要器官位置

**头部**：
- 大脑：颅腔内，控制全身活动
- 眼：眼眶内，视觉器官
- 耳：颞骨内，听觉和平衡器官
- 鼻：面部中央，呼吸和嗅觉
- 口：面部下方，进食和说话

**胸部**：
- 心脏：胸腔中部偏左，两肺之间
- 肺：胸腔两侧，左右各一
- 气管：胸腔前部，连接喉和肺
- 食管：气管后方，连接咽和胃

**腹部**：
- 胃：左上腹，食管下方
- 肝：右上腹，膈肌下方
- 胆囊：肝下方，储存胆汁
- 胰：胃后方，分泌胰液和胰岛素
- 脾：左上腹，免疫器官
- 肾：腰部脊柱两侧，左右各一
- 小肠：腹腔中部，消化吸收主要场所
- 大肠：小肠周围，吸收水分和形成粪便

### 1.2 重要生理指标与正常值

#### 1.2.1 生命体征

**体温**：
- 腋下正常值：36.0-37.2℃
- 口腔正常值：36.3-37.2℃
- 直肠正常值：36.5-37.7℃
- 体温超过37.3℃为发热
- 37.3-38℃为低热，38.1-39℃为中度发热，39.1-41℃为高热，41℃以上为超高热

**脉搏（心率）**：
- 成人安静状态下正常值：60-100次/分
- 平均约72次/分
- 老年人偏慢，女性稍快，儿童较快
- <60次/分为心动过缓，>100次/分为心动过速

**呼吸**：
- 成人安静状态下：12-20次/分
- 呼吸与脉搏之比约为1:4
- <12次/分为呼吸过缓，>20次/分为呼吸过速

**血压**：
- 收缩压（高压）正常值：90-139mmHg
- 舒张压（低压）正常值：60-89mmHg
- 理想血压：<120/80mmHg
- 正常高值：120-139/80-89mmHg
- 高血压：≥140/90mmHg
- 低血压：<90/60mmHg

#### 1.2.2 血液常规指标

**红细胞相关**：
- 红细胞计数（RBC）：男4.0-5.5×10¹²/L，女3.5-5.0×10¹²/L
- 血红蛋白（Hb）：男120-160g/L，女110-150g/L
- 红细胞压积（HCT）：男40-50%，女35-45%

**白细胞相关**：
- 白细胞计数（WBC）：4.0-10.0×10⁹/L
- 中性粒细胞：50-70%
- 淋巴细胞：20-40%
- 单核细胞：3-8%
- 嗜酸粒细胞：0.5-5%
- 嗜碱粒细胞：0-1%

**血小板相关**：
- 血小板计数（PLT）：100-300×10⁹/L

#### 1.2.3 生化指标

**血糖**：
- 空腹血糖：3.9-6.1mmol/L
- 餐后2小时血糖：<7.8mmol/L
- 糖尿病诊断：空腹≥7.0mmol/L或餐后2小时≥11.1mmol/L

**血脂**：
- 总胆固醇（TC）：<5.2mmol/L
- 甘油三酯（TG）：<1.7mmol/L
- 低密度脂蛋白胆固醇（LDL-C）：<3.4mmol/L
- 高密度脂蛋白胆固醇（HDL-C）：>1.0mmol/L

**肝功能**：
- 谷丙转氨酶（ALT）：0-40U/L
- 谷草转氨酶（AST）：0-40U/L
- 总胆红素（TBIL）：3.4-17.1μmol/L
- 白蛋白（ALB）：35-55g/L

**肾功能**：
- 肌酐（Cr）：男53-106μmol/L，女44-97μmol/L
- 尿素氮（BUN）：2.9-8.2mmol/L
- 尿酸（UA）：男208-428μmol/L，女155-357μmol/L

#### 1.2.4 其他生理指标

**体重指数（BMI）**：
- 计算公式：BMI = 体重（kg）÷ 身高²（m²）
- 正常范围：18.5-23.9
- 偏瘦：<18.5
- 超重：24-27.9
- 肥胖：≥28

**心率变异性**：
- 反映心脏自主神经功能
- 运动后心率恢复情况是健康指标之一

**血氧饱和度**：
- 正常：95-100%
- <95%为血氧降低
- <90%为低氧血症，需立即就医

---

### 【本章小结】
了解人体基本结构和生理常识是陪诊师的基础知识。人体由细胞、组织、器官、系统组成，有九大系统协同工作。掌握生命体征、血常规、生化等重要生理指标的正常值，能够帮助我们更好地理解患者的健康状况和检查结果。

### 【思考问题】
1. 你能准确说出心脏、肝脏、胃、肾这些重要器官的位置吗？
2. 正常的血压、体温、心率范围是多少？超过什么范围需要警惕？
3. BMI是如何计算的？你的BMI在正常范围内吗？

## 第二章 常见疾病认知与预防

### 2.1 循环系统常见疾病

#### 2.1.1 高血压

**什么是高血压**：
- 血压持续升高超过正常范围
- 是最常见的慢性病之一
- 被称为"无声的杀手"，早期可能无明显症状

**高血压的分级**：
- 1级高血压（轻度）：140-159/90-99mmHg
- 2级高血压（中度）：160-179/100-109mmHg
- 3级高血压（重度）：≥180/110mmHg

**常见症状**：
- 头晕、头痛
- 颈项僵硬
- 疲劳、心悸
- 视力模糊
- 鼻出血

**危险因素**：
- 遗传因素
- 高盐饮食
- 肥胖
- 长期精神紧张
- 过量饮酒
- 缺乏运动

**并发症**：
- 脑卒中（中风）
- 心肌梗死
- 心力衰竭
- 慢性肾病
- 视网膜病变

**预防与日常注意**：
- 低盐饮食（每天<5g盐）
- 控制体重
- 规律运动
- 戒烟限酒
- 保持心情舒畅
- 定期监测血压
- 遵医嘱服药，不擅自停药

#### 2.1.2 冠心病

**什么是冠心病**：
- 冠状动脉粥样硬化性心脏病
- 冠状动脉狭窄或阻塞导致心肌缺血缺氧
- 是中老年人最常见的心脏病

**常见类型**：
- 心绞痛
- 心肌梗死
- 无症状心肌缺血
- 缺血性心肌病
- 猝死

**典型症状**：
- 胸痛（压榨感、紧缩感、烧灼感）
- 胸闷、气短
- 疼痛可放射至左肩、左臂、下颌、背部
- 劳累或情绪激动时诱发
- 休息或含服硝酸甘油可缓解

**危险因素**：
- 不可改变：年龄、性别、遗传
- 可改变：高血压、高血脂、糖尿病、吸烟、肥胖、缺乏运动

**急救要点**：
- 立即停止活动，就地休息
- 舌下含服硝酸甘油
- 拨打120急救电话
- 保持呼吸道通畅
- 有条件吸氧

**预防措施**：
- 控制血压、血脂、血糖
- 戒烟限酒
- 合理饮食，低油低盐
- 适度运动，避免剧烈运动
- 保持情绪稳定
- 冬季注意保暖
- 定期体检

#### 2.1.3 心律失常

**什么是心律失常**：
- 心脏跳动的频率或节律异常
- 过快、过慢或不规则

**常见类型**：
- 早搏（房早、室早）
- 心动过速
- 心动过缓
- 房颤（心房颤动）
- 传导阻滞

**症状表现**：
- 心悸、心慌
- 心跳漏拍感
- 头晕、乏力
- 胸闷、气短
- 严重时可出现晕厥

**注意事项**：
- 避免过度劳累和情绪激动
- 不饮浓茶、咖啡、酒精等刺激性饮品
- 规律作息，不熬夜
- 定期做心电图检查
- 遵医嘱服药

---

### 2.2 内分泌代谢常见疾病

#### 2.2.1 糖尿病

**什么是糖尿病**：
- 胰岛素分泌不足或作用缺陷导致血糖升高
- 慢性代谢性疾病

**分型**：
- 1型糖尿病：胰岛素绝对缺乏，多见于青少年，需终身注射胰岛素
- 2型糖尿病：胰岛素抵抗为主，多见于中老年人，占90%以上
- 妊娠糖尿病：妊娠期间发生

**典型症状（三多一少）**：
- 多饮：喝水多
- 多食：吃得多
- 多尿：尿得多
- 体重减少：体重下降

**诊断标准**：
- 空腹血糖≥7.0mmol/L
- 餐后2小时血糖≥11.1mmol/L
- 随机血糖≥11.1mmol/L伴症状

**并发症**：
- 急性：酮症酸中毒、高渗昏迷
- 慢性：糖尿病肾病、糖尿病视网膜病变（失明）、糖尿病足（截肢）、神经病变、心血管疾病

**日常管理"五驾马车"**：
1. **饮食控制**：控制总热量，合理搭配
2. **运动治疗**：适度运动，控制体重
3. **药物治疗**：遵医嘱用药
4. **血糖监测**：定期监测血糖
5. **健康教育**：学习糖尿病知识

**注意事项**：
- 饮食定时定量
- 避免空腹运动
- 随身携带糖果，防止低血糖
- 定期检查眼底、肾功能、足部
- 不随意停药或调整剂量

#### 2.2.2 高脂血症

**什么是高脂血症**：
- 血液中脂质成分异常升高
- 包括高胆固醇血症、高甘油三酯血症

**血脂指标解读**：
- 总胆固醇（TC）：理想<5.2mmol/L
- 甘油三酯（TG）：理想<1.7mmol/L
- 低密度脂蛋白（LDL-C）："坏胆固醇"，越低越好
- 高密度脂蛋白（HDL-C）："好胆固醇"，越高越好

**危害**：
- 动脉粥样硬化
- 冠心病
- 脑梗塞
- 脂肪肝
- 胰腺炎（甘油三酯极高时）

**生活方式干预**：
- 控制饮食：减少饱和脂肪和胆固醇摄入
- 增加运动：每周至少150分钟中等强度运动
- 控制体重：BMI维持在18.5-23.9
- 戒烟限酒
- 定期复查血脂

### 2.3 呼吸系统常见疾病

#### 2.3.1 感冒与流感

**普通感冒**：
- 病原体：鼻病毒、冠状病毒等
- 症状：鼻塞、流涕、咽痛、咳嗽、低热
- 传染性：较弱
- 病程：5-7天

**流行性感冒（流感）**：
- 病原体：流感病毒
- 症状：高热、头痛、肌肉酸痛、乏力
- 传染性：强，易爆发流行
- 病程：7-14天
- 并发症：肺炎、心肌炎等，严重可致死

**区别要点**：
- 流感全身症状重，呼吸道症状轻
- 普通感冒呼吸道症状重，全身症状轻
- 流感常有高热，普通感冒多为低热或不发热

**预防措施**：
- 接种流感疫苗（每年秋季）
- 勤洗手
- 避免去人群密集场所
- 室内通风
- 增强体质

#### 2.3.2 慢性阻塞性肺疾病（COPD）

**什么是COPD**：
- 慢性支气管炎和肺气肿的总称
- 以持续性气流受限为特征
- 进行性发展，不可逆

**主要症状**：
- 慢性咳嗽
- 咳痰
- 气短或呼吸困难
- 喘息和胸闷
- 晚期出现体重下降、食欲减退

**危险因素**：
- 吸烟（最主要）
- 职业粉尘和化学物质
- 空气污染
- 感染
- 遗传因素

**日常注意**：
- 戒烟（最重要）
- 避免接触刺激性气体
- 预防感冒
- 家庭氧疗（必要时）
- 呼吸功能锻炼
- 加强营养

### 2.4 消化系统常见疾病

#### 2.4.1 胃炎与消化性溃疡

**急性胃炎**：
- 病因：不洁饮食、药物、酒精、应激
- 症状：上腹痛、恶心、呕吐、食欲不振

**慢性胃炎**：
- 病因：幽门螺杆菌感染、饮食不规律、药物
- 症状：上腹不适、饱胀、嗳气、反酸

**消化性溃疡**：
- 胃溃疡：餐后痛
- 十二指肠溃疡：空腹痛、夜间痛，进食后缓解
- 并发症：出血、穿孔、幽门梗阻、癌变

**日常注意**：
- 规律饮食，定时定量
- 避免辛辣、刺激性食物
- 戒烟酒
- 避免服用损伤胃黏膜的药物
- 保持心情舒畅

#### 2.4.2 脂肪肝

**什么是脂肪肝**：
- 肝细胞内脂肪堆积过多
- 轻者无症状，重者可发展为肝硬化

**分型**：
- 酒精性脂肪肝：长期过量饮酒
- 非酒精性脂肪肝：肥胖、糖尿病、高血脂

**症状**：
- 多数无症状，体检发现
- 部分可有乏力、右上腹不适
- 严重时出现黄疸、肝功能异常

**干预措施**：
- 戒酒
- 控制体重
- 调整饮食结构
- 增加运动
- 控制血糖、血脂
- 定期复查肝功能和B超

---

### 2.5 骨关节常见疾病

#### 2.5.1 骨质疏松

**什么是骨质疏松**：
- 骨量减少，骨组织微结构破坏
- 骨脆性增加，易发生骨折

**高危人群**：
- 绝经后女性
- 老年人
- 缺乏运动者
- 钙和维生素D摄入不足者
- 长期服用激素者

**常见症状**：
- 腰酸背痛
- 身高变矮、驼背
- 易骨折（髋部、脊柱、腕部）

**预防措施**：
- 补钙：成人每日800-1000mg
- 补充维生素D：促进钙吸收
- 适度运动：增加骨密度
- 多晒太阳
- 预防跌倒

#### 2.5.2 骨关节炎

**什么是骨关节炎**：
- 关节软骨退行性变
- 多见于中老年人
- 好发于膝关节、髋关节、脊柱等负重关节

**症状**：
- 关节疼痛
- 活动后加重，休息后缓解
- 关节僵硬（晨起明显）
- 关节活动受限
- 关节肿胀、畸形

**日常注意**：
- 控制体重，减轻关节负担
- 避免长时间站立、爬楼、爬山
- 注意关节保暖
- 适度运动（游泳、骑车等对关节友好的运动）
- 必要时使用拐杖

## 第三章 常见检查项目解读

### 3.1 实验室检查

#### 3.1.1 血常规检查

**检查目的**：
- 检查血液中各种细胞的数量和形态
- 辅助诊断感染、贫血、血液系统疾病等

**主要项目解读**：

**白细胞计数（WBC）**：
- 升高：细菌感染、炎症、应激、白血病
- 降低：病毒感染、化疗、放疗、自身免疫病

**中性粒细胞百分比（NEUT%）**：
- 升高：细菌感染、急性炎症
- 降低：病毒感染、某些药物

**淋巴细胞百分比（LYMPH%）**：
- 升高：病毒感染、淋巴细胞白血病
- 降低：免疫缺陷、激素治疗

**红细胞计数（RBC）、血红蛋白（Hb）**：
- 降低：贫血
- 升高：红细胞增多症、脱水

**血小板计数（PLT）**：
- 升高：感染、出血后、血小板增多症
- 降低：血小板减少性紫癜、化疗、再生障碍性贫血

**陪诊师注意**：
- 血常规结果需要结合临床症状综合判断
- 轻度异常不一定有病理意义
- 一次异常需要复查确认
- 有疑问建议咨询医生，不要自行诊断

#### 3.1.2 生化检查

**肝功能检查**：

**谷丙转氨酶（ALT）、谷草转氨酶（AST）**：
- 升高：肝细胞损伤（肝炎、脂肪肝、药物性肝损伤、酒精性肝病）
- AST/ALT比值>2提示酒精性肝病可能

**胆红素（TBIL、DBIL）**：
- 升高：黄疸、胆道梗阻、肝细胞损伤
- 直接胆红素升高为主：梗阻性黄疸
- 间接胆红素升高为主：溶血性黄疸

**白蛋白（ALB）**：
- 降低：营养不良、肝硬化、肾病综合征、慢性消耗性疾病

**肾功能检查**：

**肌酐（Cr）**：
- 升高：肾功能不全
- 受肌肉量影响，老年人、消瘦者可能偏低

**尿素氮（BUN）**：
- 升高：肾功能不全、高蛋白饮食、消化道出血、脱水

**尿酸（UA）**：
- 升高：痛风、肾功能不全、高嘌呤饮食、肿瘤

**血糖检查**：

**空腹血糖（GLU）**：
- 3.9-6.1mmol/L：正常
- 6.1-7.0mmol/L：空腹血糖受损（糖尿病前期）
- ≥7.0mmol/L：糖尿病（需重复确认）

**血脂检查**：

**总胆固醇（TC）、甘油三酯（TG）**：
- 升高：高脂血症
- TG极高时需警惕胰腺炎风险

**低密度脂蛋白胆固醇（LDL-C）**：
- "坏胆固醇"，与动脉粥样硬化关系最密切
- 根据危险分层有不同的控制目标

**高密度脂蛋白胆固醇（HDL-C）**：
- "好胆固醇"，有保护作用
- 降低是心血管病危险因素

#### 3.1.3 其他实验室检查

**尿常规检查**：
- 尿蛋白：阳性提示肾病可能
- 尿糖：阳性提示糖尿病可能
- 尿红细胞：阳性提示血尿（结石、感染、肾炎等）
- 尿白细胞：阳性提示尿路感染
- 尿酮体：阳性提示酮症（糖尿病酮症、饥饿）

**大便常规+潜血**：
- 白细胞：肠道感染
- 红细胞：下消化道出血
- 潜血阳性：上消化道出血（需排除饮食因素）

**肿瘤标志物**：
- 甲胎蛋白（AFP）：肝癌
- 癌胚抗原（CEA）：消化道肿瘤
- CA199：胰腺癌、胆道肿瘤
- CA125：卵巢癌
- 前列腺特异性抗原（PSA）：前列腺癌
- 注意：肿瘤标志物升高≠肿瘤，正常≠排除肿瘤，需结合影像和临床

### 3.2 影像学检查

#### 3.2.1 X线检查

**什么是X线检查**：
- 利用X射线穿透人体，不同组织密度不同，在胶片上形成黑白对比影像
- 最常用、最基础的影像学检查

**常用检查部位**：
- 胸部X线（胸片）：检查肺、心脏、肋骨
- 骨骼X线：检查骨折、关节病变、骨肿瘤
- 腹部平片：检查肠梗阻、结石、消化道穿孔

**检查前注意**：
- 摘除检查部位的金属物品（项链、首饰、皮带扣等）
- 胸片需要深吸气后屏气
- 孕妇禁做X线检查
- 儿童检查需保护性腺

**常见异常表现**：
- 肺部阴影：炎症、结核、肿瘤
- 肺纹理增粗：支气管炎、吸烟
- 骨折：骨皮质不连续
- 心影增大：心脏扩大

#### 3.2.2 CT检查

**什么是CT检查**：
- 计算机断层扫描
- X线围绕人体旋转扫描，计算机重建出断层图像
- 分辨率比X线高，能发现更小的病变

**常用检查部位**：
- 头颅CT：脑出血、脑梗塞、脑肿瘤、外伤
- 胸部CT：肺癌、肺炎、肺结核、纵隔病变
- 腹部CT：肝、胆、胰、脾、肾等腹部脏器病变
- 脊柱CT：椎间盘突出、骨折、脊柱病变

**分类**：
- 平扫CT：普通扫描，不用造影剂
- 增强CT：静脉注射造影剂，使病变更清晰，有助于鉴别诊断

**检查前注意**：
- 摘除金属物品
- 增强CT需要空腹4-6小时
- 检查前告知碘过敏史
- 肾功能不全者慎做增强CT
- 孕妇禁做CT
- 检查后多饮水，促进造影剂排出

**优缺点**：
- 优点：分辨率高、检查速度快、适用范围广
- 缺点：有辐射，比X线剂量大

#### 3.2.3 核磁共振（MRI）检查

**什么是MRI**：
- 利用磁共振现象成像
- 无电离辐射，对人体无害

**常用检查部位**：
- 头颅MRI：脑肿瘤、脑血管病、脊髓病变（比CT更清晰）
- 脊柱MRI：椎间盘突出、脊髓病变、脊柱肿瘤
- 关节MRI：韧带、半月板、软骨损伤
- 腹部MRI：肝胆胰脾病变，特别适合肝脏

**检查前注意**：
- 绝对禁忌：体内有金属植入物（心脏起搏器、人工耳蜗、金属异物等）
- 摘除所有金属物品（假牙、首饰、皮带、手表等）
- 检查时间较长（15-60分钟），需要保持静止
- 噪音较大，可戴耳塞
- 幽闭恐惧症患者可能不适

**优缺点**：
- 优点：无辐射、软组织分辨率高、多平面成像
- 缺点：检查时间长、费用高、有金属禁忌、噪音大

#### 3.2.4 B超检查

**什么是B超**：
- 利用超声波反射成像
- 无辐射、无创、便捷、费用低

**常用检查部位**：
- 腹部B超：肝、胆、胰、脾、肾
- 泌尿系B超：肾、输尿管、膀胱、前列腺
- 妇科B超：子宫、卵巢
- 产科B超：胎儿检查
- 心脏彩超：心脏结构和功能
- 血管彩超：颈动脉、四肢血管

**检查前注意**：
- 腹部B超：空腹8小时以上
- 泌尿系、妇科B超：需要憋尿（膀胱充盈）
- 经阴道B超：需要排空膀胱

**优缺点**：
- 优点：无辐射、无创、便捷、费用低、可实时观察
- 缺点：受气体干扰，图像清晰度有限，依赖操作者经验

### 3.3 其他检查

#### 3.3.1 心电图检查

**什么是心电图**：
- 记录心脏电活动的图形
- 无创、便捷、常用

**检查目的**：
- 诊断心律失常
- 诊断心肌缺血、心肌梗死
- 了解心脏扩大、肥厚
- 监测药物对心脏的影响
- 评估起搏器功能

**检查前注意**：
- 避免剧烈运动
- 放松，避免紧张
- 保持安静
- 不要空腹（可能低血糖影响结果）

**常见异常**：
- 窦性心动过速/过缓
- 早搏（房早、室早）
- 房颤
- ST-T改变（提示心肌缺血可能）
- 传导阻滞

**陪诊师注意**：
- 心电图是即时的，只反映检查当时的情况
- 心电图异常不等于有心脏病
- 心电图正常也不能完全排除心脏病
- 需结合症状和其他检查综合判断

#### 3.3.2 内镜检查

**胃镜**：
- 检查食管、胃、十二指肠
- 适应症：上腹痛、反酸、吞咽困难、黑便、呕血、贫血
- 检查前：空腹8-12小时
- 无痛胃镜：静脉麻醉，睡一觉就做完，需有人陪同
- 检查后：2小时后再进食，无痛胃镜当天不能开车

**结肠镜**：
- 检查结肠、直肠
- 适应症：便血、腹痛、腹泻、便秘、大便习惯改变、筛查肠癌
- 检查前：肠道准备（喝泻药清肠），检查前1-2天少渣饮食
- 检查后：注意有无腹痛、便血

## 第四章 常用药物基本知识

### 4.1 药物基本知识

#### 4.1.1 药物的基本概念

**什么是药物**：
- 用于预防、治疗、诊断疾病，有目的地调节人体生理功能的物质
- 规定有适应症、用法和用量

**药物的分类**：

**按来源分**：
- 天然药物：中药、植物药、动物药
- 化学药物：化学合成的药物
- 生物药物：疫苗、抗体、细胞因子等

**按处方管理分**：
- 处方药（Rx）：必须凭医生处方才能购买和使用
- 非处方药（OTC）：不需要处方，可自行购买使用，相对安全

**按给药途径分**：
- 口服给药：片剂、胶囊、颗粒、口服液
- 注射给药：静脉、肌肉、皮下注射
- 局部给药：外用软膏、滴眼剂、滴鼻剂、栓剂
- 吸入给药：气雾剂、雾化吸入

#### 4.1.2 药物的作用和不良反应

**药物的作用**：
- 治疗作用：达到防治疾病目的的作用
- 不良反应：与治疗目的无关，给患者带来不适或痛苦的反应

**常见不良反应类型**：

**副作用**：
- 治疗剂量下出现的与治疗目的无关的反应
- 一般较轻微，可耐受
- 如感冒药引起嗜睡、降压药引起头晕

**毒性反应**：
- 剂量过大或用药时间过长引起的严重不良反应
- 对机体有明显损害
- 如肝毒性、肾毒性、耳毒性

**过敏反应**：
- 免疫系统对药物的异常反应
- 与剂量无关，与体质有关
- 表现：皮疹、瘙痒、发热、哮喘，严重时过敏性休克

**后遗效应**：
- 停药后血药浓度已降至有效浓度以下仍残存的药理效应
- 如服用安眠药后第二天早晨仍有困倦感

**停药反应**：
- 突然停药后原有疾病加剧
- 如长期服用降压药突然停药血压反跳

#### 4.1.3 影响药物作用的因素

**药物方面**：
- 剂量：剂量越大作用越强，但毒性也可能越大
- 剂型：不同剂型吸收速度不同
- 给药途径：静脉>吸入>肌肉>皮下>口服>直肠>皮肤

**机体方面**：
- 年龄：儿童和老年人对药物更敏感
- 性别：女性对某些药物更敏感
- 体重：体重大者可能需要较大剂量
- 病理状态：肝肾功能不全影响药物代谢排泄
- 心理因素：安慰剂效应

**饮食方面**：
- 空腹服药吸收快
- 饭后服药吸收慢但对胃刺激小
- 某些食物影响药物吸收（如钙影响四环素吸收）
- 酒精影响药物代谢

### 4.2 常用药物分类及代表药

#### 4.2.1 心血管系统药物

**降压药**：

**1. 钙通道阻滞剂（CCB）**：
- 代表药：硝苯地平、氨氯地平、非洛地平
- 特点：降压效果明确，适用范围广
- 副作用：头痛、面部潮红、脚踝水肿

**2. 血管紧张素转换酶抑制剂（ACEI）**：
- 代表药：卡托普利、依那普利、贝那普利
- 特点：对心肾有保护作用，适合合并糖尿病、肾病患者
- 副作用：干咳（发生率约20%）、血管性水肿

**3. 血管紧张素Ⅱ受体拮抗剂（ARB）**：
- 代表药：氯沙坦、缬沙坦、厄贝沙坦
- 特点：与ACEI类似，但无干咳副作用
- 副作用：较少，偶见头晕

**4. β受体阻滞剂**：
- 代表药：美托洛尔、比索洛尔
- 特点：减慢心率，适合合并冠心病、心衰患者
- 副作用：心动过缓、乏力、支气管痉挛

**5. 利尿剂**：
- 代表药：氢氯噻嗪、吲达帕胺、呋塞米
- 特点：降压温和，常与其他药合用
- 副作用：电解质紊乱、尿酸升高、血糖影响

**抗心绞痛药**：
- 硝酸甘油：舌下含服，急救用，缓解心绞痛急性发作
- 硝酸异山梨酯（消心痛）：预防心绞痛发作
- 美托洛尔：减少心肌耗氧

**调脂药**：
- 他汀类：阿托伐他汀、瑞舒伐他汀，主要降胆固醇
- 贝特类：非诺贝特，主要降甘油三酯

**抗血小板药**：
- 阿司匹林：预防血栓，心脑血管病二级预防
- 氯吡格雷：阿司匹林不能耐受或需要更强抗血小板时用

#### 4.2.2 内分泌系统药物

**口服降糖药**：

**1. 双胍类**：
- 代表药：二甲双胍
- 特点：2型糖尿病首选，不增加体重，不引起低血糖
- 副作用：胃肠道反应，乳酸酸中毒（罕见）

**2. 磺脲类**：
- 代表药：格列本脲、格列美脲、格列齐特
- 特点：促进胰岛素分泌，降糖作用强
- 副作用：低血糖、体重增加

**3. α-葡萄糖苷酶抑制剂**：
- 代表药：阿卡波糖、伏格列波糖
- 特点：延缓碳水化合物吸收，降低餐后血糖
- 副作用：腹胀、排气增多

**4. DPP-4抑制剂**：
- 代表药：西格列汀、沙格列汀
- 特点：作用温和，低血糖风险低，不影响体重
- 副作用：较少

**胰岛素**：
- 1型糖尿病必须用
- 2型糖尿病口服药控制不好时用
- 必须皮下注射
- 最大副作用：低血糖

**甲状腺疾病用药**：
- 左甲状腺素钠（优甲乐）：甲状腺功能减退替代治疗，需终身服药
- 甲巯咪唑（他巴唑）：治疗甲状腺功能亢进

#### 4.2.3 呼吸系统药物

**止咳药**：
- 中枢性镇咳：右美沙芬（无成瘾性）、可待因（有成瘾性）
- 外周性镇咳：那可丁

**祛痰药**：
- 氨溴索
- 乙酰半胱氨酸
- 羧甲司坦

**平喘药**：
- β受体激动剂：沙丁胺醇（万托林，急救用）、特布他林
- 茶碱类：氨茶碱、茶碱缓释片
- 抗胆碱药：异丙托溴铵
- 激素吸入剂：布地奈德、氟替卡松（控制气道炎症）

**感冒药**：
- 多为复方制剂，包含解热镇痛、抗过敏、缩血管、镇咳等成分
- 注意不要重复用药，避免过量

#### 4.2.4 消化系统药物

**抑酸药**：
- 质子泵抑制剂（PPI）：奥美拉唑、兰索拉唑、雷贝拉唑，抑酸作用最强
- H2受体拮抗剂：法莫替丁、雷尼替丁，抑酸作用中等

**胃黏膜保护剂**：
- 硫糖铝
- 枸橼酸铋钾
- 铝碳酸镁（达喜）：同时有中和胆汁作用

**促胃动力药**：
- 多潘立酮（吗丁啉）
- 莫沙必利
- 伊托必利

**泻药**：
- 容积性泻药：乳果糖、聚乙二醇，安全，适合老年人和长期用
- 刺激性泻药：番泻叶、大黄、酚酞，作用强，不建议长期用
- 润滑性泻药：开塞露、甘油灌肠剂，局部用，适合急性便秘

#### 4.2.5 解热镇痛抗炎药

**对乙酰氨基酚（扑热息痛、泰诺林）**：
- 解热镇痛，抗炎作用弱
- 对胃刺激小
- 过量可致肝损伤，注意不要超量

**布洛芬**：
- 解热镇痛抗炎作用都有
- 对胃有一定刺激
- 胃不好的人慎用

**阿司匹林**：
- 解热镇痛
- 小剂量用于抗血小板，预防心脑血管病
- 对胃刺激大，肠溶片相对好一些
- 哮喘、胃溃疡者慎用

---

### 4.3 安全用药知识

#### 4.3.1 用药基本原则

**1. 遵医嘱用药**：
- 不要自行增减剂量
- 不要自行停药
- 不要自行换药

**2. 按时按量用药**：
- 一天3次不是三餐后，而是每8小时一次
- 定时服药，保持血药浓度稳定
- 漏服了怎么办：一般想起时立即补服，若接近下次服药时间则跳过，不要加倍

**3. 正确的给药方式**：
- 口服药：一般温水送服，不要用茶水、饮料
- 肠溶片、缓释片、控释片：不要掰开、嚼碎
- 舌下含服：硝酸甘油，不要吞服
- 外用软膏：按说明使用，注意用量和疗程

**4. 注意饮食对药物的影响**：
- 有些药需要空腹吃（如吗丁啉、奥美拉唑）
- 有些药需要饭后吃（如对胃有刺激的药）
- 有些药不能与某些食物同服（如头孢不能喝酒）

#### 4.3.2 老年人用药注意事项

**老年人用药特点**：
- 肝肾功能减退，药物代谢排泄减慢
- 对药物敏感性增加
- 常同时患多种疾病，用药多
- 记忆力减退，容易漏服或错服

**老年人用药原则**：
- 受益原则：确实需要才用药
- 五种药物原则：同时用药不超过5种
- 小剂量原则：从小剂量开始，逐渐增加
- 择时原则：根据时间生物学选择最佳给药时间
- 暂停用药原则：怀疑不良反应及时停药观察

**特别提醒**：
- 不要自行购买"特效药"、"偏方"
- 不要轻信广告和病友推荐
- 定期清理药箱，过期药及时丢弃
- 用药前仔细阅读说明书
- 有疑问及时咨询医生或药师

#### 4.3.3 常见用药误区

**误区1：是药三分毒，能不吃就不吃**
- 正确观点：该用的药必须用，如高血压、糖尿病的药不能随便停
- 不用药的后果比药物副作用严重得多

**误区2：副作用大的药不是好药**
- 正确观点：副作用是药品的固有属性，不一定会发生
- 医生选择药物是权衡利弊的结果
- 出现副作用及时就医，不要自行停药

**误区3：别人用着好的药我也能用**
- 正确观点：每个人情况不同，适合别人的不一定适合你
- 用药要个体化，不能盲目跟风

**误区4：中药没有副作用**
- 正确观点：是药就有副作用，中药也不例外
- 很多中药有肝毒性、肾毒性
- 中药也需要在医生指导下使用

**误区5：进口药、贵药就是好药**
- 正确观点：适合自己的才是好药
- 很多国产药效果很好，价格便宜
- 不要盲目迷信进口药

## 第五章 化验单阅读入门

### 5.1 化验单阅读的基本原则

#### 5.1.1 正确认识参考值

**什么是参考值**：
- 参考值（正常值）是指健康人群中某项指标的数值范围
- 通常取95%可信区间，意味着有5%的健康人结果也可能在范围外
- 不同实验室、不同检测方法的参考值可能不同

**正确解读参考值**：
- 在参考值范围内不一定完全正常
- 超出参考值范围不一定有病
- 轻度异常可能没有临床意义
- 需要结合临床症状和其他检查综合判断
- 动态变化比单次结果更重要

#### 5.1.2 影响检验结果的因素

**分析前因素（占检验误差的60-70%）**：

**患者因素**：
- 年龄、性别
- 饮食：空腹状态、饮食成分
- 运动：剧烈运动可引起多项指标变化
- 情绪：紧张、激动
- 体位：立位与卧位结果不同
- 月经周期、妊娠
- 吸烟、饮酒

**标本因素**：
- 标本采集时间
- 标本采集部位
- 抗凝剂使用
- 标本溶血、脂血
- 标本保存和运输

**分析中因素（占检验误差的10-20%）**：
- 仪器性能
- 试剂质量
- 操作人员技术

**分析后因素（占检验误差的10-20%）**：
- 结果审核
- 报告书写
- 结果解读

#### 5.1.3 化验单阅读的基本方法

**第一步：看清基本信息**：
- 患者姓名、性别、年龄
- 送检科室、诊断
- 检验日期、时间
- 标本类型

**第二步：逐项查看结果**：
- 结果数值
- 与参考值对比
- 标注异常（↑升高、↓降低、H高、L低）
- 注意单位

**第三步：综合分析**：
- 相关指标联合分析
- 与历史结果对比
- 结合临床症状
- 必要时咨询医生

---

### 5.2 常见化验单解读实例

#### 5.2.1 血常规化验单解读

**血常规报告示例**：

| 项目 | 结果 | 参考值 | 异常 |
|------|------|--------|------|
| 白细胞计数（WBC） | 12.5×10⁹/L | 4.0-10.0 | ↑ |
| 中性粒细胞百分比（NEUT%） | 78% | 50-70 | ↑ |
| 淋巴细胞百分比（LYMPH%） | 18% | 20-40 | ↓ |
| 中性粒细胞绝对值（NEUT#） | 9.75×10⁹/L | 2.0-7.0 | ↑ |
| 淋巴细胞绝对值（LYMPH#） | 2.25×10⁹/L | 0.8-4.0 | - |
| 红细胞计数（RBC） | 4.5×10¹²/L | 4.0-5.5 | - |
| 血红蛋白（Hb） | 135g/L | 120-160 | - |
| 血小板计数（PLT） | 220×10⁹/L | 100-300 | - |

**解读**：
- 白细胞和中性粒细胞升高，提示可能存在细菌感染
- 淋巴细胞相对降低是因为中性粒细胞升高了
- 红细胞、血红蛋白、血小板正常
- 结合患者如有发热、咳嗽等症状，支持感染诊断

**另一例血常规报告**：

| 项目 | 结果 | 参考值 | 异常 |
|------|------|--------|------|
| 白细胞计数（WBC） | 3.5×10⁹/L | 4.0-10.0 | ↓ |
| 血红蛋白（Hb） | 95g/L | 120-160 | ↓ |
| 血小板计数（PLT） | 85×10⁹/L | 100-300 | ↓ |

**解读**：
- 三系均降低（白细胞、红细胞/血红蛋白、血小板都低）
- 需要警惕：再生障碍性贫血、骨髓增生异常综合征、白血病、脾功能亢进等
- 必须进一步检查，建议血液科就诊

#### 5.2.2 生化化验单解读

**生化报告示例（肝功能异常）**：

| 项目 | 结果 | 参考值 | 异常 |
|------|------|--------|------|
| 谷丙转氨酶（ALT） | 125U/L | 0-40 | ↑ |
| 谷草转氨酶（AST） | 85U/L | 0-40 | ↑ |
| 总胆红素（TBIL） | 25μmol/L | 3.4-17.1 | ↑ |
| 白蛋白（ALB） | 38g/L | 35-55 | - |
| γ-谷氨酰转肽酶（GGT） | 95U/L | 10-60 | ↑ |

**解读**：
- 转氨酶明显升高，提示肝细胞损伤
- 可能原因：病毒性肝炎、脂肪肝、酒精性肝病、药物性肝损伤
- GGT升高明显，酒精性肝病可能性大
- 需要进一步检查：乙肝五项、丙肝抗体、肝脏B超等

**生化报告示例（肾功能异常）**：

| 项目 | 结果 | 参考值 | 异常 |
|------|------|--------|------|
| 肌酐（Cr） | 156μmol/L | 53-106 | ↑ |
| 尿素氮（BUN） | 10.5mmol/L | 2.9-8.2 | ↑ |
| 尿酸（UA） | 520μmol/L | 208-428 | ↑ |
| 估算肾小球滤过率（eGFR） | 52ml/min | >90 | ↓ |

**解读**：
- 肌酐、尿素氮升高，提示肾功能不全
- eGFR 52ml/min，属于CKD 3期（中度肾功能损害）
- 尿酸同时升高，可能是高尿酸血症导致肾损害，也可能是肾排尿酸能力下降
- 需要查明原因，肾内科就诊，保护肾功能

#### 5.2.3 血脂化验单解读

**血脂报告示例**：

| 项目 | 结果 | 参考值 | 异常 | 危险分层目标 |
|------|------|--------|------|--------------|
| 总胆固醇（TC） | 6.8mmol/L | <5.2 | ↑ | - |
| 甘油三酯（TG） | 2.5mmol/L | <1.7 | ↑ | - |
| 低密度脂蛋白胆固醇（LDL-C） | 4.2mmol/L | <3.4 | ↑ | 极高危<1.8，高危<2.6 |
| 高密度脂蛋白胆固醇（HDL-C） | 0.9mmol/L | >1.0 | ↓ | - |

**解读**：
- 典型的高脂血症表现
- LDL-C升高是最主要的心血管危险因素
- HDL-C降低也是危险因素
- 生活方式干预+药物治疗
- LDL-C目标值根据心血管危险分层确定，不是简单看参考值

### 5.3 常见检验误区与注意事项

#### 5.3.1 常见认识误区

**误区1：指标异常就是有病**
- 很多轻度异常是生理性的或检验误差
- 一次异常不能确诊，需要复查
- 参考值是95%范围，5%的健康人可能在范围外

**误区2：指标正常就是没病**
- 很多疾病早期指标可能正常
- 有些疾病影响的指标不在常规检查范围内
- 体检正常不代表完全健康，有症状还是要就医

**误区3：肿瘤标志物升高就是癌症**
- 肿瘤标志物特异性不高
- 很多良性疾病也会升高
- 升高需要进一步检查，但不要恐慌
- 正常也不能完全排除肿瘤

**误区4：不同医院的结果可以直接比较**
- 不同实验室的检测方法、仪器、试剂可能不同
- 参考值可能不同
- 最好在同一家医院复查，便于比较

#### 5.3.2 陪诊师的注意事项

**能做的**：
- 帮患者查看结果是否有异常
- 向患者解释基本的医学常识
- 提醒患者哪些异常需要重视
- 帮助患者整理历史结果，便于对比
- 协助患者向医生提问

**不能做的**：
- 不能"解读"结果给出诊断
- 不能说"你这个就是XX病"
- 不能建议患者用什么药
- 不能说"这个没事，不用看医生"
- 不能仅凭化验单就下结论

**正确的说法**：
- "这个指标有点高/低，具体意义需要让医生看看"
- "这个异常可能和XX有关，不过得医生结合你的情况判断"
- "这个指标比上次好/不好了，一会儿问问医生"
- "有几个异常，等下让医生重点看看"

化验单阅读是陪诊师必备的实用技能。本章介绍了化验单阅读的基本原则，包括正确认识参考值、了解影响检验结果的因素、掌握基本阅读方法，通过血常规、生化、血脂等常见化验单的实例解读，帮助学员掌握基本解读能力，同时明确了陪诊师在化验单解读中的注意事项和边界。

## 第六章 医疗知识应用案例

### 案例一：高血压患者就医全程协助

#### 案例背景
王先生，58岁，发现高血压5年，平时服药不规律，血压控制不佳，最近经常头晕。女儿在外地，帮他预约了专家号，请陪诊师小李陪同就诊。

#### 陪诊前准备
**了解患者情况**：
- 高血压病史5年
- 目前服用硝苯地平缓释片，但是想起来才吃，经常漏服
- 平时偶尔测血压，大概160/95mmHg
- 最近一周头晕明显，特别是早上起床时
- 没有定期复查

**准备资料**：
- 整理患者正在服用的药物清单
- 记录最近几次血压测量值
- 列出患者想要咨询医生的问题：
  1. 为什么血压总是控制不好？
  2. 头晕是不是高血压引起的？
  3. 是否需要换药？
  4. 平时需要注意什么？

#### 就诊过程协助
**见医生前**：
- 提醒患者早上空腹（可能需要抽血）
- 到医院后先测血压（165/98mmHg）
- 候诊时和患者沟通，确认没有遗漏的问题

**见医生时**：
- 协助患者向医生描述病情：
  "医生，我发现高血压5年了，现在吃硝苯地平缓释片，但经常忘了吃。平时血压大概160/95，最近一周早上起来总是头晕。"
- 补充患者容易遗漏的信息：
  "他女儿说他经常漏服药物，有时候两三天都不吃。"
- 记录医生的建议：
  1. 调整药物：氨氯地平5mg每天一次，比索洛尔2.5mg每天一次
  2. 每天早晚各测一次血压，记录下来
  3. 低盐饮食，每天盐摄入量不超过5g
  4. 一个月后复查

**就诊后**：
- 取药时核对药品名称和剂量
- 用大号字体在药盒上标注用法："每天早上1片，每天1次"
- 帮患者设置手机服药提醒
- 详细讲解注意事项：
  "王叔叔，医生说您主要是服药不规律导致血压控制不好。现在换成两种药，每天早上一起吃，可别忘了。这是我给您写的服药时间表，您贴在冰箱上，每天吃完打个勾。每天早晚各测一次血压，记在这个本子上，下次复查带给医生看。"

#### 后续跟进
- 第二天电话随访，确认患者已按时服药
- 提醒患者监测血压
- 一周后再次随访，了解血压控制情况和有无不适
- 提前提醒患者复查时间

#### 案例点评
1. **充分的诊前准备是服务质量的基础**
2. **协助患者清晰、完整地向医生描述病情很重要**
3. **准确记录医嘱，用患者容易理解的方式传达**
4. **用实际行动帮助患者落实医嘱（如设置提醒、制作记录表格）**
5. **诊后跟进体现服务的完整性和专业性**

### 案例二：糖尿病患者化验结果解读与沟通

#### 案例背景
张阿姨，62岁，2型糖尿病10年，平时服用二甲双胍和格列美脲，但是饮食控制不好，也很少监测血糖。最近体检发现血糖很高，还有蛋白尿，女儿很担心，陪她来内分泌科就诊，请陪诊师小王陪同。

#### 化验单解读准备
**整理化验单**：
- 空腹血糖：10.5mmol/L（正常3.9-6.1）↑
- 糖化血红蛋白（HbA1c）：9.2%（正常<6.5%）↑
- 尿常规：尿蛋白（++）↑
- 尿微量白蛋白/肌酐比值：350mg/g（正常<30）↑
- 肌酐：112μmol/L（正常44-97）↑
- 血脂：胆固醇6.5mmol/L，甘油三酯2.8mmol/L，LDL-C 4.2mmol/L

**准备问题清单**：
- 血糖控制不好怎么办？
- 尿蛋白是不是说明肾有问题了？
- 要不要打胰岛素？
- 饮食具体要注意什么？

#### 协助沟通与解读
**向医生描述情况**：
"医生，我得糖尿病10年了，现在吃二甲双胍和格列美脲。平时不太忌口，也很少测血糖。这次体检发现血糖很高，还有尿蛋白，您看看是不是肾出问题了？"

**记录医生医嘱**：
1. **血糖方面**：
   - 加用DPP-4抑制剂西格列汀
   - 购买血糖仪，每天监测血糖并记录
   - 目标：空腹<7mmol/L，餐后<10mmol/L，HbA1c<7%

2. **肾脏方面**：
   - 尿蛋白阳性提示糖尿病肾病早期
   - 加用ARB类药物（既降压又降尿蛋白）
   - 控制血压<130/80mmHg
   - 定期复查肾功能和尿蛋白

3. **血脂方面**：
   - 加用他汀类调脂药
   - LDL-C目标<1.8mmol/L（糖尿病属于极高危）

4. **生活方式**：
   - 严格糖尿病饮食，控制主食量
   - 适当运动，每周至少150分钟
   - 控制体重

#### 向患者解释与宣教
用通俗的语言向张阿姨解释：

**关于血糖**：
"阿姨，您现在糖化血红蛋白9.2%，说明最近三个月平均血糖大概11mmol/L左右，确实太高了。正常应该控制在7%以下。就像您家里的温度计，空腹血糖是即时温度，糖化血红蛋白是三个月平均温度。"

**关于尿蛋白**：
"尿蛋白两个加号说明肾确实有点受影响了，但现在还是早期，及时治疗可以延缓进展，不用太担心。就像水管刚开始漏水，及时修补还来得及。"

**关于饮食**：
"医生说您主要是饮食控制不好。我给您举个例子：您原来一顿吃两碗饭，现在改成一碗半；原来爱吃红烧肉，现在改成一周吃一次，每次吃两三块；粥和稀饭升糖快，尽量少喝，改成喝牛奶或豆浆。"

**关于监测**：
"我帮您在手机上设了闹钟，每天早中晚饭后测血糖，记在这个本子上，下次复查带给医生看。就像写日记一样，把每天的血糖记下来，医生才能根据情况调整用药。"

#### 案例点评
1. **用比喻和生活化的语言解释医学问题，患者更容易理解**
2. **糖尿病管理需要细致、具体的指导，而不是空泛的说教**
3. **陪诊师不仅要陪同就诊，更要帮助患者理解和落实医嘱**
4. **健康教育是陪诊服务的重要增值部分**

### 案例三：冠心病患者紧急情况处理

#### 案例背景
李先生，65岁，冠心病病史3年，去年放了心脏支架。今天来医院常规复查，陪诊师小张陪同。在等待叫号的时候，李先生突然说胸口不舒服。

#### 紧急情况处理
**第一步：立即评估**：
小张立即询问："李叔叔，您怎么不舒服？是胸痛吗？什么样的疼？"
李先生说："胸口闷，像压了块石头，有点喘不上气。"
小张观察：李先生面色苍白，额头有汗，手捂着胸口。

**第二步：立即采取措施**：
1. 立即扶李先生坐下，保持安静
2. 从李先生口袋里拿出硝酸甘油，协助舌下含服
3. 立即去叫医生和护士
4. 给李先生家属打电话告知情况

**第三步：配合医生处理**：
医生很快赶来，立即给李先生做心电图，提示心肌缺血表现。
医生说："先含服硝酸甘油，观察5分钟，如果不缓解需要紧急处理。"
小张在旁边协助：
- 安抚李先生情绪："李叔叔您别紧张，医生来了，没事的，放松，深呼吸。"
- 监测李先生的脉搏和面色
- 协助护士给李先生吸氧
- 5分钟后李先生胸痛缓解

**第四步：后续安排**：
医生建议李先生住院进一步检查治疗。
小张协助：
1. 办理急诊住院手续
2. 联系家属告知住院安排
3. 整理李先生的随身物品和病历资料
4. 安慰家属："您别太担心，现在李叔叔已经缓解了，住院进一步检查一下，有什么情况我随时和您联系。"

#### 经验总结
**紧急情况处理要点**：
1. **保持冷静，不要慌乱**：你的情绪会影响患者
2. **第一时间寻求专业帮助**：不要自己处理，找医生护士
3. **同时采取必要措施**：如协助含服硝酸甘油、让患者安静休息
4. **安抚患者情绪**：紧张会加重心肌缺血
5. **及时联系家属**：告知情况和安排

**陪诊师应具备的能力**：
1. 识别常见急症表现
2. 掌握基本急救知识
3. 具备应急处理能力
4. 良好的沟通协调能力

**重要提醒**：
- 陪诊师不是医生，不要尝试超出能力范围的处理
- 立即寻求专业医疗帮助是第一位的
- 你的角色是协助和配合，而不是独立处理

### 案例四：老年人多重用药整理与指导

#### 案例背景
王奶奶，78岁，患高血压、糖尿病、冠心病、骨关节炎等多种疾病，每天要吃十几种药。经常搞不清什么药什么时候吃，有时候漏服，有时候重复吃，家里药堆得乱七八糟。子女请陪诊师小周帮忙整理药物，陪同就诊调整用药。

#### 药物整理过程
**第一步：收集所有药物**：
小周让王奶奶把所有正在吃的药都拿出来，包括处方药、非处方药、保健品。
共收集到15种：
- 降压药3种
- 降糖药2种
- 抗血小板药1种
- 调脂药1种
- 胃药2种
- 止痛片2种
- 钙片、维生素D、鱼油等保健品4种

**第二步：制作药物清单**：

| 药物名称 | 规格 | 用法 | 适应症 | 备注 |
|----------|------|------|--------|------|
| 氨氯地平 | 5mg | 每天1次，早上 | 高血压 | |
| 缬沙坦 | 80mg | 每天1次，早上 | 高血压、冠心病 | |
| 美托洛尔 | 25mg | 每天2次，早晚 | 高血压、冠心病 | |
| 二甲双胍 | 0.5g | 每天3次，餐后 | 糖尿病 | |
| 格列美脲 | 2mg | 每天1次，早餐前 | 糖尿病 | |
| 阿司匹林 | 100mg | 每天1次，晚上 | 冠心病 | |
| 阿托伐他汀 | 20mg | 每天1次，晚上 | 调脂 | |
| 奥美拉唑 | 20mg | 每天1次，早上 | 胃不好 | 自行购买 |
| 铝碳酸镁 | 0.5g | 每天3次，餐后 | 胃痛 | 自行购买 |
| 布洛芬 | 0.2g | 疼痛时吃 | 关节痛 | 经常吃 |
| 对乙酰氨基酚 | 0.5g | 疼痛时吃 | 关节痛 | 两种止痛药交替吃 |

**第三步：发现问题**：
1. **重复用药**：两种胃药作用类似，可能不必要；两种止痛药同时使用增加副作用风险
2. **药物相互作用**：布洛芬会影响降压药和阿司匹林的效果，增加胃肠道出血风险
3. **用药时间不明确**：有些药应该饭前吃，有些饭后吃，王奶奶都一起吃
4. **自行购买药物多**：很多药是王奶奶自己在药店买的，没有医生指导

**第四步：陪同就诊调整用药**：
小周带着整理好的药物清单陪同王奶奶看老年科医生。
医生看了清单后说：
"老人家吃的药确实太多了，我们来精简一下。"

医生调整方案：
1. 奥美拉唑和铝碳酸镁选一种，症状明显时临时用铝碳酸镁，不需要长期吃两种
2. 停掉布洛芬，只用对乙酰氨基酚，减少对胃和肾脏的负担
3. 二甲双胍改成缓释片，一天一次，减少服药次数
4. 保健品可以继续吃，但不要指望有治疗作用

**第五步：用药指导与管理**：
小周根据调整后的方案，重新整理药物：

**制作服药时间表**：

| 时间 | 药物 |
|------|------|
| 早餐前 | 格列美脲1片 |
| 早餐后 | 氨氯地平1片、缬沙坦1片、美托洛尔1片、二甲双胍缓释片1片 |
| 晚餐后 | 美托洛尔1片 |
| 睡前 | 阿司匹林1片、阿托伐他汀1片、钙片1片 |
| 必要时 | 铝碳酸镁1-2片（胃痛时）、对乙酰氨基酚1片（关节痛时） |

**购买分装药盒**：
小周帮王奶奶买了一个一周分装药盒，分成早中晚睡前四格。
"王奶奶，我帮您把一周的药都分装好，您每天按格子吃，这样就不会吃错了。我每周日过来帮您装下一周的药。"

**设置提醒**：
在王奶奶的老年手机上设置了四个服药时间的闹钟提醒。

**定期随访**：
小周每周上门一次，检查服药情况，有问题及时联系医生。

#### 案例点评
1. **老年人多重用药是普遍问题，需要专业的用药管理**
2. **陪诊师可以通过药物整理、制作清单和时间表，帮助老年人规范用药**
3. **发现潜在的药物问题，及时建议医生调整，避免药物不良反应**
4. **定期随访和持续管理，确保用药方案的落实**

## 【课程总结】

本课程系统介绍了陪诊师需要掌握的医疗常识，包括人体基本结构与生理常识、常见疾病认知与预防、常见检查项目解读、常用药物基本知识、化验单阅读入门，并通过4个真实案例展示了医疗知识在实际陪诊服务中的应用。

**核心要点回顾**：
1. 掌握基本生理指标正常值，理解人体基本结构和功能
2. 了解高血压、糖尿病、冠心病等常见疾病的基本知识和注意事项
3. 熟悉血常规、生化、影像等常见检查的目的和意义
4. 掌握常用药物的分类、作用和注意事项
5. 学会基本的化验单阅读方法，知道哪些异常需要重视
6. 医疗知识的应用要注意边界，不能超出陪诊师的执业范围

**重要提醒**：
陪诊师不是医生，我们学习医疗知识的目的是：
1. 更好地理解患者的病情和需求
2. 更好地协助医患沟通
3. 更好地解释和传达医嘱
4. 及时发现异常情况，提醒患者就医
5. 绝对不能自行诊断、推荐用药、解读检查结果给出结论

**下一步行动建议**：
1. 制作一份常用生理指标正常值速查表，方便服务时查阅
2. 整理一份常见疾病基本知识点清单，定期复习
3. 准备一个小本子，服务中遇到不懂的医学问题及时记录，事后查阅或请教专业人士
4. 服务中涉及医疗问题时，始终保持谦虚谨慎的态度，建议患者咨询医生
